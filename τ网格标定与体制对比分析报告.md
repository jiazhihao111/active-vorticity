## 4. 根因分析

### 4.1 核心发现：均匀平坦化塌缩

```
训练初期 (rf_loss ≈ 12):
  phys_curv: 较大     flex+: 较大     flex-: 较大
  ↓
训练中期 (rf_loss ≈ 1.3):
  phys_curv: 0.10     flex+: 0.101    flex-: 0.103  ← H-flex SUPPORT!
  ↓
训练收敛 (rf_loss ≈ 0.16):
  phys_curv: 0.04     flex+: 0.04     flex-: 0.04   ← H-flex INCONCLUSIVE
```

**模型学到的最优策略**: 将所有转移（物理+柔性, 正例+负例）统一压扁到近零曲率。

### 4.2 损失函数的结构性问题

`rigid_flexible_layered_loss` 的公式 (R):

```
Λ = λ_phys · ‖F_phys‖² + λ_flex · max(0, ‖F_flex‖² - τ)
```

两个问题:

1. **R-惩罚主导**: 物理层无阈值惩罚对所有 token 生效，且与柔性层惩罚同向（都鼓励平坦）。推离信号 (铁律九 push) 相对太弱。

2. **铁律九被压制**: 
   - `push_phys = F.relu(margin - curv)` — 仅在 curv < margin 时激活
   - `push_flex = F.relu(tau - curv)` — 仅在 curv < τ 时激活
   - 当 curv 已接近 0 时（被 R-惩罚压缩），推离信号本身也趋近零

3. **柔性层缺少定向分化信号**: 公式(R)对柔性正例和柔性负例的惩罚是相同的（只要 curv² > τ 就惩罚），没有将正例推向合法带内、负例推向带外的定向力。

### 4.3 回环检出失效

在所有体制下，`loop_count = 0.00`。回环检出依赖 `GaugeField.loop_back_holonomy_flatness` 中的阈值比较，说明当前离散回环检测方法无法捕捉叙事闭环——叙事闭合是语义层面的，而非 token 位置的几何自交。

## 5. τ 标定结论

| 维度 | 结论 |
|------|------|
| **τ 对训练结果的影响** | 几乎为零。rf_loss 变异系数仅 0.06 |
| **τ 对诊断的影响** | 无。信号强度 (SNR≈0.007) 不足以产生任何判决差异 |
| **推荐的标定方法** | τ 不能通过纯数学优化标定，需要从数据分布估计：τ = μ_flex + k·σ_flex，否则会退化到零 |
| **H-rigid 结论** | **验证通过**: 物理层近零曲率 (0.04 << 0.2)，铁律八成立 |
| **H-flex 结论** | **未验证**: 柔性层无法产生差异化信号，需要重新设计损失函数 |

## 6. 改进建议

### 6.1 损失函数重构 (短期)

将当前"统一压缩 + 弱推离"改为"定向分化":

```python
# 新公式: 定向分化损失
L_flex = λ_pos · (curv_pos - target_low)²         # 正例拉到合法带内
       + λ_neg · max(0, target_high - curv_neg)²  # 负例推到带外  
       + λ_phys · ‖F_phys‖²                       # 物理保持零曲率
```

关键变化:
- 正例有明确目标位置 (target_low)，而非仅阈值惩罚
- 负例被明确推向远离目标 (target_high)
- 取消统一 R-惩罚，改为按例分类的定向力

### 6.2 回环检测改进 (中期)

- 用语义嵌入空间的余弦相似度替代离散位置自交检测
- 或直接放弃 token-level 回环，改用故事级闭合的连续度量

### 6.3 数据增强 (中期)

- 增加柔性违规的强度（当前 neg 偏离太小）
- 引入软违规模板（接近但不越过阈值边界的负例）
