## 零、一句话结论

**LLM隐状态动力学是非平衡态有源约束系统：需要持续驱动力（F_active = α*·ḣ）维持语义轨道运动，因果约束力（F_c）将轨迹约束在合法流形上。扣除驱动力后，合法轨迹的约束力做功P_c ≈ 0。**

## 二、v8.0数学形式化

### 2.1 运动方程

$$m\ddot{\mathbf{h}}(t) + \gamma\dot{\mathbf{h}}(t) = \underbrace{\alpha^*\dot{\mathbf{h}}(t)}_{\text{轨道维持力}} + \underbrace{\mathbf{F}_c(\mathbf{h}, \dot{\mathbf{h}})}_{\text{因果约束力}} + \boldsymbol{\xi}(t)$$

等价形式：

$$m\ddot{\mathbf{h}}(t) + (\gamma - \alpha^*)\dot{\mathbf{h}}(t) = \mathbf{F}_c(\mathbf{h}, \dot{\mathbf{h}}) + \boldsymbol{\xi}(t)$$

### 2.2 三项力的物理含义

| 力 | 表达式 | 物理含义 | 实验证据 |
|----|--------|---------|---------|
| 轨道维持力 | $\mathbf{F}_{active} = \alpha^*\dot{\mathbf{h}}$ | 对抗LayerNorm归一化收缩，维持隐状态在语义轨道上运动 | α*≈1.17, P_active占总功率~97% |
| 因果约束力 | $\mathbf{F}_c = m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} - \alpha^*\dot{\mathbf{h}}$ | 将轨迹约束在合法因果流形上 | pos P_c≈0.37≈0, neg P_c≈1.09>0 |
| 随机噪声 | $\boldsymbol{\xi}(t)$ | Token离散化跳跃的随机涨落 | 未单独量化 |

### 2.3 修正约束力做功

$$P_c(t) = \mathbf{F}_c \cdot \dot{\mathbf{h}} = (m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} - \alpha^*\dot{\mathbf{h}}) \cdot \dot{\mathbf{h}}$$

**v8.0核心可证伪预测**：

- 合法轨迹：$P_c(t) \approx 0$（约束力垂直于速度，不做功）
- 非法轨迹：$P_c(t) > 0$（约束力做正功，轨迹偏离约束流形）

### 3.2 能量分解

| 分量 | 正例均值 | 负例均值 | 占比(pos) |
|------|----------|----------|-----------|
| P_raw (v7.0总功率) | 14.08 | 15.89 | 100% |
| P_active (轨道维持力做功) | 13.71 | 14.80 | **97.4%** |
| P_constraint (约束力做功) | 0.37 | 1.09 | **2.6%** |

**解读**：系统97%的做功来自轨道维持力，仅2.6%来自约束力。v7.0看到的"P(t)>>0"几乎完全由F_active贡献。

### 3.3 α*的物理来源

| 指标 | 数值 |
|------|------|
| α*全局最优 | ≈1.1 |
| α*局部估计(逐轨迹) | mean=1.17, std=0.064 |
| α*范围 | [1.06, 1.27] |
| base_projection σ_max | 0.982 |
| base_projection 条件数 | 3.14 |

**α*≈1.17的物理解释**：

α*不是摩擦系数，而是**轨道维持力的有效强度**。其值略大于1的可能原因：

1. **Transformer残差连接的累积效应**：2层Transformer，每层残差连接贡献约0.5的增量，累积≈1.0
2. **LayerNorm的归一化收缩**：LayerNorm将隐状态范数压缩，需要额外驱动力补偿，贡献约0.1-0.2
3. **base_projection的范数保持**：σ_max≈0.982≈1.0，投影几乎保持范数，不显著影响α*

### 3.4 隐状态范数演化（负阻尼修正）

| 指标 | 正例 | 负例 |
|------|------|------|
| 起始范数 | 3.96 | 3.97 |
| 终止范数 | 3.80 | 3.80 |
| 增长率 | **-4.1%** | **-4.1%** |
| 速度范数 | 3.31 | 3.37 |
| 加速度范数 | 4.94 | 5.05 |

**关键修正**：隐状态范数**随时间步衰减**（-4%），而非增长。

**"负阻尼→自加速"的解读被否证**。正确解读：

- $\gamma_{eff} = \gamma - \alpha^* < 0$ 不意味着系统自加速
- α*·ḣ 是**轨道维持力**，对抗LayerNorm的归一化收缩
- 系统处于**非平衡态定态**（NESS）：需要持续能量输入维持运动，但宏观性质稳定
- 类比：恒流电路中的电子——需要持续电场驱动才能维持电流，但电子速度不无限增长（受散射平衡）

### 3.5 Proxy B（残差和）vs Proxy D（α·ḣ）

| 指标 | 结果 |
|------|------|
| Pearson相关 | **-0.42** (负相关) |
| 余弦相似度 | **-0.38** (方向相反) |
| 范数比(res/vel) | 1.12 (量级接近) |

**结论**：Proxy B（Transformer层残差和）与Proxy D（α*·ḣ）**不等价**。

残差和的方向与速度方向**相反**（cos≈-0.38），说明Transformer层的残差输出不是"推动"隐状态前进的力，而是"修正/抑制"力——类似于阻尼或纠偏。F_active = α*·ḣ 的物理来源**不是**残差和，而是更底层的自回归生成机制。

### 3.8 约束方程隐式形式

F_c⊥ḣ 且 rank(F_c)≈rank(vel)，约束力存在于速度的零空间中：

$$C_i(\mathbf{h}, \dot{\mathbf{h}}) = \mathbf{n}_i^T \dot{\mathbf{h}} = 0, \quad i = 1, ..., d - r$$

其中n_i是约束流形法向量，r是速度有效秩，d-r是约束方程数量。这是**非完整约束**的几何特征。

## 四、v8.0的物理图像

### 4.1 非平衡态有源约束系统

```
                    ┌──────────────────────┐
                    │   LayerNorm归一化     │
                    │   (收缩力, 范数↓)     │
                    └──────────┬───────────┘
                               │ 对抗
                    ┌──────────▼───────────┐
                    │  F_active = α*·ḣ     │
                    │  (轨道维持力, 范数→)  │
                    └──────────┬───────────┘
                               │ 约束
                    ┌──────────▼───────────┐
                    │  F_c (因果约束力)     │
                    │  (方向约束, P_c≈0)    │
                    └──────────┬───────────┘
                               │ 涨落
                    ┌──────────▼───────────┐
                    │  ξ(t) (Token跳跃噪声) │
                    └──────────────────────┘
```

### 4.2 三种力的类比

| 力 | 电路类比 | 流体类比 | 语义含义 |
|----|---------|---------|---------|
| F_active=α*·ḣ | 恒压源驱动电流 | 压力梯度驱动流体 | 自回归目标驱动生成 |
| F_c | 电阻约束电流路径 | 管道壁约束流向 | 因果逻辑约束叙事 |
| ξ(t) | 热噪声 | 布朗运动 | Token离散化随机性 |

### 4.3 为什么合法轨迹P_c≈0

合法叙事遵循因果逻辑，其隐状态演化"顺应"了约束流形的几何结构。约束力F_c垂直于速度方向（不做功），轨迹在约束流形上"惯性滑行"——但这个"惯性"需要F_active持续供能才能维持。

非法叙事违反因果逻辑，其隐状态演化"逆着"约束流形，约束力F_c与速度方向有夹角（做正功），轨迹偏离约束流形。

## 五、有效秩反向的物理解释

v7.0实验发现：pos有效秩=7 > neg有效秩=6。

**旧直觉（被否证）**：合法轨迹受约束→维度更低→有效秩更小。

**v8.0新解释**：因果约束不是"降维打击"，而是**"高维流形上的定向导流"**。

- 合法叙事包含多维度的因果、情感、事实交织，速度向量张成的切空间**信息密度极高**（有效秩=7）
- 非法叙事（幻觉/车轱辘话）往往陷入局部的重复或单一的逻辑死胡同，速度向量高度共线，**有效秩反而低**（=6）

**类比**：高速公路（合法）比乡间小路（非法）需要更多车道（更高维），但车流更高效（P_c≈0）。
