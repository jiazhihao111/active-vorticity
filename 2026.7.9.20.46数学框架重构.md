## 三、 新数学体系重构：六步闭环作业（调用 模块4 · 科学研究版）

### 第0步：直觉触发
- **核心动作**：接收 MVE 实验的“裸事实”——合法轨迹二阶导数（加速度）更小，空间度量平坦。
- **探索方向**：寻找一种不依赖空间弯曲，仅依靠“力（势场梯度）”和“惯性”就能区分合法与非法路径的数学框架。

### 第一步：解构剥离
- **无效节点清单**：黎曼度量张量 $G_{\mu\nu}$、规范连接 $A_\mu$、陈数、纤维丛结构、高斯-博内定理。
- **核心因果链**：离散 Token 生成 $\rightarrow$ 隐状态在平直空间中转移 $\rightarrow$ 受到因果势场 $V(h)$ 的吸引与排斥 $\rightarrow$ 形成具有特定加速度特征的轨迹。

### 第二步：拓扑映射
- **拓扑不变量**：**语义作用量（Semantic Action）**。合法叙事路径是使语义作用量取极值的路径。
- **可证伪预测**：如果理论为真，合法轨迹的“动能（速度平方）”与“势能”应存在守恒或平滑转换关系，而非法轨迹会出现能量（作用量）的剧烈耗散或突变。

### 第三步：极简重构（设定底层规则集）
构建 **GUIT-TRT v6.0（分析力学版）** 的 5 条底层公理：
1. **平直基底公理**：语义空间 $\mathcal{H}$ 是配备标准欧几里得度量的 $\mathbb{R}^d$。（解释 MVE 中度量坍缩至 $I$）。
2. **因果势场公理**：存在标量势函数 $V: \mathbb{R}^d \rightarrow \mathbb{R}$，事实锚点为势谷，逻辑谬误区为势垒。
3. **语义惯性公理**：上下文依赖度表现为语义质量 $m$，抵抗状态的剧烈改变。
4. **耗散与涨落公理**：离散 Token 的跳跃等效于热力学摩擦 $\gamma$ 与高斯白噪声 $\xi(t)$。
5. **最小作用量公理**：合法生成路径遵循带耗散的欧拉-拉格朗日方程。

**核心运动方程（替代原 F-损失）**：
$$ m \ddot{\mathbf{h}}(t) + \gamma \dot{\mathbf{h}}(t) + \nabla V(\mathbf{h}(t)) = \boldsymbol{\xi}(t) $$
*(注：合法路径沿势谷运动，$\nabla V \approx 0$，故加速度 $\ddot{\mathbf{h}}$ 极小，轨迹平坦；非法路径强穿势垒，$\nabla V$ 极大，导致 $\ddot{\mathbf{h}}$ 剧烈震荡。完美闭环 MVE 实验现象！)*

### 第四步：演化（纯粹涌现）
- **涌现行为记录**：在平直空间中运行该随机微分方程（SDE），观察是否自然涌现“合法轨迹平滑、非法轨迹毛刺”的宏观现象。
- **未预期副效应**：可能会涌现“极限环（重复性车轱辘话）”或“混沌吸引子（严重幻觉）”，需记录其对应的势场 $V(h)$ 拓扑缺陷。

### 第五步：参数迭代
- **偏差归因**：若模型仍无法区分正负例，归因于势场 $V(h)$ 的参数化表达能力不足，或摩擦系数 $\gamma$ 过大导致系统过阻尼（失去惯性特征）。
- **调整方案**：引入 Neural ODE 或 SINDy（稀疏非线性动力学识别）来逆向拟合 $V(h)$。

## 一、 六步闭环作业执行流程（工程设计版）

### 第0步：直觉触发与数据准备
- **核心动作**：构建包含“合法事实演进”与“逻辑断裂/幻觉”的平行语料库，提取对应的隐状态轨迹。
- **数据规格**：
  - 正样本（Pos）：逻辑严密、事实准确的叙事序列（如高质量维基百科段落）。
  - 负样本（Neg）：在特定节点强行注入逻辑谬误或事实错误的序列。
  - 提取目标：模型倒数第二层的隐状态序列 $\mathbf{H} = [\mathbf{h}_1, \mathbf{h}_2, ..., \mathbf{h}_N]$，维度通常为 $d=768$ 或 $4096$。

### 第一步：解构剥离（降噪与降维）
- **核心动作**：剔除离散 Token 生成带来的高频内禀噪声，将高维黑盒映射到低维连续动力学空间。
- **执行规范**：
  1. **时序平滑**：使用 Savitzky-Golay 滤波器或高斯过程（GP）对 $\mathbf{H}$ 进行平滑，得到连续可导轨迹 $\tilde{\mathbf{H}}$。
  2. **流形降维**：使用非线性降维（如 UMAP 或基于自编码器的潜空间投影）将 $\tilde{\mathbf{H}}$ 降至 $k$ 维（建议 $k \in [16, 32]$），得到潜变量序列 $\mathbf{Z}(t)$。
- **信息论避坑**：⚠️ **区分精度缺失**风险。若平滑窗口过大，会抹平真实的因果转折（势垒）；若过小，则残留离散噪声。需通过频谱分析确定截止频率。

### 第二步：拓扑映射（动力学方程构建）
- **核心动作**：将降维后的轨迹映射为经典力学中的过阻尼朗之万方程（Overdamped Langevin Dynamics）。
- **方程设定**：假设语义惯性在宏观尺度上被高摩擦耗散主导，系统演化为：
  $$ \gamma \dot{\mathbf{z}}(t) = -\nabla V(\mathbf{z}(t)) + \boldsymbol{\xi}(t) $$
  其中 $\gamma$ 为摩擦系数，$V(\mathbf{z})$ 为待求标量势场，$\boldsymbol{\xi}(t)$ 为高斯白噪声。

### 第三步：定向重构（算法选择与损失设计）
- **核心动作**：构建神经网络参数化势场 $V_\theta(\mathbf{z})$，并设计优化目标。
- **算法推荐**：**Neural ODE 势场拟合**（比 SINDy 更适合高维非线性潜空间）。
- **网络架构**：
  - 输入：潜状态 $\mathbf{z} \in \mathbb{R}^k$。
  - 输出：标量能量值 $V_\theta(\mathbf{z}) \in \mathbb{R}$。
  - 梯度计算：利用自动微分（AutoGrad）计算保守力 $\mathbf{F}_\theta(\mathbf{z}) = -\nabla_\mathbf{z} V_\theta(\mathbf{z})$。
- **M-损失函数（Mechanics Loss）**：
  $$ \mathcal{L} = \underbrace{\frac{1}{N}\sum_{t} \left\| \gamma \dot{\mathbf{z}}(t) - \mathbf{F}_\theta(\mathbf{z}(t)) \right\|^2}_{\text{动力学重构误差}} + \underbrace{\lambda_{\text{smooth}} \|\nabla^2 V_\theta\|_F^2}_{\text{势场平滑正则化}} + \underbrace{\lambda_{\text{contrast}} \max(0, m - (V_\theta(\mathbf{z}_{neg}) - V_\theta(\mathbf{z}_{pos})))}_{\text{势垒对比损失}} $$

### 第四步：定向演化（训练与涌现记录）
- **核心动作**：执行梯度下降，监控势场地貌的涌现过程。
- **关键监控指标**：
  1. **重构误差收敛曲线**：判断神经网络是否成功拟合了轨迹的宏观走向。
  2. **势垒高度 $\Delta V$**：记录正负样本在势场中的能量差，验证是否形成有效的因果隔离。
- **副效应记录**：若出现 $V_\theta(\mathbf{z})$ 剧烈震荡（非物理的“毛刺”），说明网络过拟合了残余噪声，需增大 $\lambda_{\text{smooth}}$。

### 第五步：参数迭代与实证锚定
- **核心动作**：基于训练结果调节超参数，输出最终势场。
- **参数调节逻辑**：
  - 若 $\Delta V$ 不显著（正负样本能量混杂）：增大 $\lambda_{\text{contrast}}$，或检查降维步骤是否丢失了关键的因果特征。
  - 若动力学重构误差无法下降：引入动量项（恢复二阶微分方程 $\ddot{\mathbf{z}}$），测试系统是否处于欠阻尼状态。

```python
import torch
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy import stats
from scipy.signal import savgol_filter

class ConstraintPowerAnalyzer:
    """
    GUIT-TRT v7.0: 约束力做功分析器
    核心假说: 合法轨迹 P(t)≈0 (理想约束不做功), 非法轨迹 P(t)<<0 (约束破裂做负功)
    """
    
    def __init__(self, mass=1.0, friction=0.1, window_length=11, polyorder=3):
        self.m = mass          # 语义质量 (可标定)
        self.gamma = friction  # 摩擦系数 (可标定)
        self.window_length = window_length
        self.polyorder = polyorder
        
    def compute_derivatives(self, H_smooth):
        """
        计算一阶和二阶导数 (中心差分法)
        输入: H_smooth [B, T, D]
        输出: dH_dt [B, T, D], d2H_dt2 [B, T, D]
        """
        B, T, D = H_smooth.shape
        device = H_smooth.device
        
        # 一阶导数 (速度)
        dH_dt = torch.zeros_like(H_smooth)
        dH_dt[:, 1:-1, :] = (H_smooth[:, 2:, :] - H_smooth[:, :-2, :]) / 2.0
        dH_dt[:, 0, :] = H_smooth[:, 1, :] - H_smooth[:, 0, :]
        dH_dt[:, -1, :] = H_smooth[:, -1, :] - H_smooth[:, -2, :]
        
        # 二阶导数 (加速度)
        d2H_dt2 = torch.zeros_like(H_smooth)
        d2H_dt2[:, 1:-1, :] = (H_smooth[:, 2:, :] - 2*H_smooth[:, 1:-1, :] + H_smooth[:, :-2, :])
        d2H_dt2[:, 0, :] = d2H_dt2[:, 1, :]  # 边界填充
        d2H_dt2[:, -1, :] = d2H_dt2[:, -2, :]
        
        return dH_dt, d2H_dt2
    
    def compute_constraint_power(self, H_smooth, mask=None):
        """
        计算约束力做功功率 P(t) = (m*h¨ + γ*h˙) · h˙
        输入: 
            H_smooth: [B, T, D] 平滑轨迹
            mask: [B, T] 可选，标记有效时间步 (排除边界)
        输出:
            P_t: [B, T] 功率时间序列
            F_res: [B, T, D] 残余力场 (可用于后续分析)
        """
        # 1. 计算导数
        dH_dt, d2H_dt2 = self.compute_derivatives(H_smooth)
        
        # 2. 计算残余力 F_res = m*h¨ + γ*h˙
        F_res = self.m * d2H_dt2 + self.gamma * dH_dt
        
        # 3. 计算功率 P(t) = F_res · h˙ (逐点内积)
        P_t = torch.sum(F_res * dH_dt, dim=-1)  # [B, T]
        
        # 4. 应用掩码 (排除边界效应)
        if mask is None:
            # 默认排除首尾2个点 (差分边界)
            mask = torch.ones_like(P_t)
            mask[:, :2] = 0
            mask[:, -2:] = 0
        
        P_t = P_t * mask
        
        return P_t, F_res, dH_dt, d2H_dt2
    
    def statistical_comparison(self, P_pos, P_neg, alpha=0.01):
        """
        正负样本功率分布的统计对比
        输入: P_pos, P_neg: [N_pos], [N_neg] 展平的功率值
        输出: 字典包含均值、标准差、偏度、峰度、p-value等
        """

        P_neg = P_neg[P_neg != 0].cpu().numpy()
        
        stats_dict = {}

            stats_dict[f'{prefix}_mean'] = np.mean(arr)
            stats_dict[f'{prefix}_std'] = np.std(arr)
            stats_dict[f'{prefix}_skew'] = stats.skew(arr)
            stats_dict[f'{prefix}_kurt'] = stats.kurtosis(arr)
            stats_dict[f'{prefix}_median'] = np.median(arr)
        
        # 假设检验
        # 1. Pos 均值是否接近 0 (单样本 t 检验)
        t_stat_pos, p_val_pos = stats.ttest_1samp(P_pos, 0)
        stats_dict['pos_vs_zero_p'] = p_val_pos

        t_stat_diff, p_val_diff = stats.ttest_ind(P_pos, P_neg, equal_var=False)
        stats_dict['pos_vs_neg_p'] = p_val_diff
        stats_dict['cohens_d'] = (np.mean(P_pos) - np.mean(P_neg)) / \
                                 np.sqrt((np.std(P_pos)**2 + np.std(P_neg)**2)/2)
        
        # 3. 分布形状检验 (Kolmogorov-Smirnov)
        ks_stat, ks_p = stats.ks_2samp(P_pos, P_neg)
        stats_dict['ks_stat'] = ks_stat
        stats_dict['ks_p'] = ks_p
        
        return stats_dict
```

### 一、 核心工程实现：$P(t)$ 计算与可视化 Pipeline

本脚本包含数据平滑、导数计算、功率计算与统计可视化。您可以直接将其集成到您的 MVE 实验环境中。

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.signal import savgol_filter
from scipy.stats import ks_2samp

class ConstraintPowerAnalyzer:
    def __init__(self, window_length=11, polyorder=3, gamma=1.0):
        """
        :param window_length: Savitzky-Golay 平滑窗口 (必须为奇数)
        :param polyorder: 多项式阶数 (求二阶导至少需要 3)
        :param gamma: 摩擦系数 (可通过合法轨迹能量守恒区间标定)
        """
        self.window_length = window_length
        self.polyorder = polyorder
        self.gamma = gamma

    def compute_derivatives(self, H_raw):
        """
        计算平滑后的一阶导(速度)和二阶导(加速度)
        H_raw shape: [B, T, D]
        """
        B, T, D = H_raw.shape
        H_dot = np.zeros_like(H_raw)
        H_ddot = np.zeros_like(H_raw)
        
        for b in range(B):
            for d in range(D):
                # 1. 平滑原始轨迹
                h_smooth = savgol_filter(H_raw[b, :, d], self.window_length, self.polyorder)
                # 2. 计算一阶导 (速度)
                H_dot[b, :, d] = savgol_filter(h_smooth, self.window_length, self.polyorder, deriv=1, delta=1.0)
                # 3. 计算二阶导 (加速度)
                H_ddot[b, :, d] = savgol_filter(h_smooth, self.window_length, self.polyorder, deriv=2, delta=1.0)
                
        return H_dot, H_ddot

    def compute_power(self, H_dot, H_ddot):
        """
        计算约束力做功功率 P(t) = (h_ddot + gamma * h_dot) \cdot h_dot
        """
        # 残余力 F_res = h_ddot + gamma * h_dot (假设质量 m=1)
        F_res = H_ddot + self.gamma * H_dot
        
        # 点积计算功率，沿特征维度 D 求和
        # P shape: [B, T]
        P = np.sum(F_res * H_dot, axis=-1) 
        return P

    def analyze_and_plot(self, H_pos_raw, H_neg_raw, save_path="constraint_power_analysis.png"):
        """
        执行完整分析并输出可视化报告
        """
        # 1. 计算导数
        H_pos_dot, H_pos_ddot = self.compute_derivatives(H_pos_raw)
        H_neg_dot, H_neg_ddot = self.compute_derivatives(H_neg_raw)
        
        # 2. 计算功率 P(t)
        P_pos = self.compute_power(H_pos_dot, H_pos_ddot).flatten()
        P_neg = self.compute_power(H_neg_dot, H_neg_ddot).flatten()
        
        # 3. 统计检验 (Kolmogorov-Smirnov test)
        ks_stat, p_value = ks_2samp(P_pos, P_neg)
        
        # 4. 可视化
        plt.figure(figsize=(12, 6))
        
        # 直方图与 KDE
        sns.histplot(P_pos, bins=100, kde=True, color='green', alpha=0.5, label=f'Pos (Mean: {np.mean(P_pos):.3f}, Var: {np.var(P_pos):.3f})')
        sns.histplot(P_neg, bins=100, kde=True, color='red', alpha=0.5, label=f'Neg (Mean: {np.mean(P_neg):.3f}, Var: {np.var(P_neg):.3f})')
        
        plt.axvline(0, color='black', linestyle='--', linewidth=2, label='Zero Power (Ideal Constraint)')
        plt.title(f'Constraint Force Power Distribution $P(t)$\n(KS Stat: {ks_stat:.3f}, p-value: {p_value:.2e})', fontsize=14)
        plt.xlabel('Power $P(t) = (\ddot{\mathbf{h}} + \gamma \dot{\mathbf{h}}) \cdot \dot{\mathbf{h}}$', fontsize=12)
        plt.ylabel('Density', fontsize=12)
        plt.legend()
        plt.grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig(save_path, dpi=300)
        plt.show()
        
        return P_pos, P_neg, ks_stat, p_value
```

## A2. v8.0理论升级：带非保守主动力的拉格朗日系统

### A2.1 运动方程修正

v7.0: $m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} = \mathbf{F}_c$ （假设F_c⊥v，不做功）

v8.0: $m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} = \mathbf{F}_{active} + \mathbf{F}_c$ （引入非保守主动力）

修正约束力: $\mathbf{F}_c = m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} - \mathbf{F}_{active}$

修正功率: $P_c(t) = \mathbf{F}_c \cdot \dot{\mathbf{h}} = (m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} - \mathbf{F}_{active}) \cdot \dot{\mathbf{h}}$

### A2.2 F_active的物理来源与估计

**方案D（最终采纳）**: $\mathbf{F}_{active} = \alpha \cdot \dot{\mathbf{h}}$

物理含义：v7.0中P(t) = (m·ḧ + γ·ḣ)·ḣ >> 0，如果系统被一个与速度成正比的力驱动（F ∝ v），则P = F·v ∝ ‖v‖² > 0。这正是**粘性驱动（viscous driving）**的特征——过阻尼系统中，驱动力与速度方向一致，大小成正比。

### A2.5 物理解读

LLM隐状态动力学是**过阻尼粘性驱动系统**：

$$m\ddot{\mathbf{h}} + \gamma\dot{\mathbf{h}} = \alpha^*\dot{\mathbf{h}} + \mathbf{F}_c$$

其中：
- $\alpha^* \approx 1.1$：有效驱动力系数，略大于摩擦系数γ
- $\mathbf{F}_{active} = \alpha^*\dot{\mathbf{h}}$：自回归过程的内禀驱动力
- $\mathbf{F}_c$：约束力，合法轨迹上F_c·ḣ ≈ 0

**等价表述**：$m\ddot{\mathbf{h}} + (\gamma - \alpha^*)\dot{\mathbf{h}} = \mathbf{F}_c$

即有效摩擦系数 $\gamma_{eff} = \gamma - \alpha^* < 0$——系统是**负阻尼**的！这意味着隐状态不是在耗散，而是在**自加速**——每一步都在积累动量，而非减速。
