## 七、根因诊断

### 7.1 成功因子

1. **刚性层锁零**: 刚柔并济损失函数 (公式A, λ_phys=1.0) 在数学上是正确的 — 刚性分量对所有体制一致锁零
2. **拓扑约束**: Gauss-Bonnet 代理在工程上可行 — 阻止隐流形塌缩到平庸拓扑 (τ_sem ×35.64)
3. **维度升级**: 32D → 128D 揭示了隐空间的几何结构 — 低维空间无法承载 Frenet-Serret 标架
4. **InfoNCE 推离**: 对比学习框架比铰链损失更适合叙事对比任务

### 7.2 持续困难

1. **正负例分化弱**: 合法叙事与非法叙事的几何差异在当前 MVE 尺度下极小 (Cohen's d 通常 < 0.1)
2. **FS SVD 投影噪声**: 128D → 3D 维度骤降使得 κ、τ 由噪声主导，重尾分布掩盖结构信号
3. **NPNW 世界简单性**: 网格世界的故事结构简单，排列变体产生的几何差异可能本身就不大
4. **训练深度**: 8-15 epochs 不足以让隐流形充分发育

### C. 统计方法说明

- **配对 t 检验**: 同源三变体的 paired t-test，单侧 (除以 2)
- **Cohen's d**: 效应量，d ≥ 0.2 (小), ≥ 0.5 (中), ≥ 0.8 (大)
- **Wilcoxon 符号秩检验**: 配对非参数检验，用于验证 t 检验结论
- **CV (变异系数)**: σ/μ，度量 τ/κ 的相对离散度
- **固化度**: 理论的可复现置信度，从初始先验经实验证据贝叶斯更新
