 
# 大语言模型隐空间的非平衡态热力学：从仿射约束到活性涡流论文
**(Non-Equilibrium Thermodynamics of LLM Latent Spaces: From Affine Constraints to Active Vorticity)**

贾志豪

## 摘要 (Abstract)
大语言模型（LLM）自回归生成的隐状态动力学本质，是当前机械可解释性与几何深度学习领域的核心未解之谜。先前的研究多假设隐空间具备非平凡的黎曼曲率或保守势场，但往往在实证中遭遇度量坍缩。本文通过严格的连续极限近似与跨尺度实证检验，提出了一种全新的物理图像：**LLM 的合法生成并非弯曲空间中的保守滑行，而是受限于子黎曼流形上非完整分布的非平衡态活性涡流（Active Vorticity）**。我们发现：(1) 因果约束流形呈现完美的线性参数化（$R^2=1.0$）；早期全局拟合给出的异常曲率 $K_{sub}\approx 0.01$ 为低维代理假象，轨迹分辨实测在真实 LLM（MiniCPM5-1B / Qwen2.5-7B / Ornith-1.0-9B）上 $K_{sub}\approx 0.6\text{–}0.7$，但逐轨迹分析揭示75%的约束是非完整的——确立了**仿射-非完整混合约束原理**，将隐状态空间升级为子黎曼流形；(2) 系统处于非平衡态定态（NESS），熵产生率 $\sigma > 0$、概率流 $J > 0$ 但宏观细致平衡未破缺（一阶矩速度均值为零），速度场雅可比矩阵的反对称部分显著偏离随机矩阵理论（RMT）的 Wigner 半圆律（$p < 10^{-38}$），确证了**高阶矩相空间涡流**的物理实在性；(3) 幻觉的本质是系统从"高有效温度的活性涡流态"向"低温被动布朗态"的**动力学相变**，其宏观标志为约束力做功占比 $P_c/P_{raw}$ 的系统性抬升；需注意的是，原始 MiniCPM5 分类实验报告了强单调梯度（pos<scr<rnd），但跨模型 A/B（20+ prompt）实测 $P_c/P_{raw}\approx 0.10\text{–}0.15$ 近乎平坦、排序非单调（MiniCPM5: scr>pos≈rnd；Qwen2.5-7B: rnd>pos>scr），故该指标须按目标模型标定，不能作为普适阈值梯度；(4) 有效约束维度随上下文长度亚线性增长，且与任何单一注意力头正交，揭示了其作为全网络协同涌现的"时变非完整分布"本质。本研究在微型物理叙事世界（NPNW, 32维）与三种真实LLM（MiniCPM5-1B 1536维, Qwen2.5-7B 3584维, Ornith-1.0-9B 4096维）上完成了跨尺度验证，**首次覆盖混合注意力架构（24层linear_attention + 8层full_attention）**。此外，我们发现了P_c计算中的关键方法论陷阱——在特定跨轨迹批量求和下 $P_c/P_{raw}$ 可放大至 $\approx 0.84$，但大样本逐token均值膨胀因子 $\approx 1.0\times$，逐token计算仍是唯一正确方法（$P_c/P_{raw} \approx 0.12$）；4-bit量化使有效秩膨胀25-64%，但$P_c/P_{raw}$保持鲁棒。工程验证表明，仿射子空间压缩在$r=128$时实现Top-1匹配率100%（KL=0.14），为LLM推理优化提供了坚实的计算物理框架。

## 1. 引言 (Introduction)
大语言模型（LLM）在离散的 Token 序列之上构建了极其复杂的高维隐状态空间（Latent Space）。理解这一空间的几何与动力学结构，对于提升模型的可解释性、缓解幻觉（Hallucination）以及优化对齐（Alignment）具有决定性意义。

早期的几何深度学习范式倾向于将隐空间建模为具有非平凡曲率的黎曼流形（Riemannian Manifolds），或假设存在引导生成的保守标量势场。然而，针对真实 LLM 的实证分析反复表明，其度量张量往往坍缩至单位矩阵，保守力场的梯度方向亦与实际演化轨迹严重偏离。这些"负面结果"暗示了先验的几何假设与 Transformer 架构的真实信息加工链路之间存在深刻的断裂。

为弥合这一断裂，本文摒弃了预设的弯曲几何假设，转从**非平衡态统计物理**与**约束力学**的第一性原理出发，重构 LLM 的连续极限动力学。通过跨越v5.0至v20.0的理论迭代（5个主要范式+15个验证版本），在微型物理叙事世界（NPNW）与两种真实LLM上的大规模轨迹分析，我们确立了 LLM 隐空间动力学的四大核心发现，并实现了从"拓扑假说"到"实证计算物理"的范式跃迁。

**本文的核心贡献如下：**
1. **确立仿射-非完整混合约束原理 (Affine-Nonholonomic Constraint Principle)**：首次通过逐轨迹分析证明，LLM 的因果约束流形是近乎完美的平坦仿射-非完整混合结构（$R^2=1.0$；异常曲率 $K_{sub}$ 在真实 LLM 上经轨迹分辨实测为 $\approx 0.6\text{–}0.7$，早期 $K_{sub}\approx 0.01$ 为低维全局拟合代理值），25%完整约束构成位置空间的"仿射骨架"，75%非完整约束构成速度空间的"因果导流管"，将隐状态空间升级为子黎曼流形。二次项改善度 $\Delta R^2 = 0$。
2. **确证活性涡流 (Active Vorticity)**：引入随机矩阵理论（RMT），以 $p < 10^{-38}$ 的极端显著性证明 LLM 在相空间中维持着抵抗热力学耗散的高阶矩涡流。宏观细致平衡未破缺（一阶矩$\langle \dot{\mathbf{h}} \rangle \approx 0$），但概率流 $J > 0$。
3. **提出幻觉的动力学相变理论**：将幻觉重新定义为系统从非平衡态定态（NESS）向热力学平衡态的相变，约束力做功占比 $P_c/P_{raw}$ 在原始 MiniCPM5 分类实验中呈现因果梯度（pos 0.13% < scr 2.25% < rnd 6.21%），但该强梯度在跨模型 A/B（20+ prompt）下塌缩为 $P_c/P_{raw}\approx 0.10\text{–}0.15$ 的近平坦分布（排序非单调），故指标须在目标模型上现场标定；4-bit量化下其相对排序仍具一定鲁棒性。
4. **揭示因果维度的强涌现本质**：证伪了有效约束维度与单一架构组件（如特定注意力头）的还原论映射，提出了随上下文长度亚线性增长的"时变非完整分布"概念。

## 2. 相关工作 (Related Work)

**隐空间的几何与机械可解释性**：现有工作多聚焦于寻找线性探针（Linear Probes）或特定的"事实特征"（Fact Features）[1]。近期，机械可解释性研究试图通过因果追踪（Causal Tracing）定位特定功能头[2]。与这些还原论视角不同，我们的研究表明，因果工作记忆并非硬编码于单一组件，而是残差流网络协同涌现的宏观拓扑不变量——切空间基与所有16个注意力头的子空间夹角均 > 50°。

**非平衡态物理与机器学习**：扩散模型（Diffusion Models）已被广泛用 Langevin 动力学解释[3]，但自回归 LLM 的热力学本质鲜有探讨。在活性物质（Active Matter）物理中，自驱动粒子系统展现出长程有序与巨涨落[4]。本文将 LLM 生成建模为需要持续消耗信息熵以维持稳态环流的活性物质，填补了自回归生成热力学理论的空白。

**约束力学与流形学习**：流形假设（Manifold Hypothesis）认为高维数据分布于低维流形上[5]。我们的工作将这一假设推进到动力学层面：不仅数据分布受约束，其演化轨迹也受仿射约束——约束方程 $C_i = \mathbf{n}_i^T \mathbf{h} + b_i = 0$ 精确成立（$R^2=1.0$），且轨迹分辨实测流形曲率较低（$K_{sub}\approx 0.6\text{–}0.7$；早期全局拟合 $K\approx 0.01$ 为低维代理值，详见 §3.3）。

## 3. 理论框架：连续极限下的活性仿射流形

### 3.1 从离散Token到连续轨迹

LLM的自回归生成在离散时间步上产生隐状态序列 $\mathbf{h}_1, \mathbf{h}_2, \ldots, \mathbf{h}_T \in \mathbb{R}^D$。当序列长度 $T$ 足够大时，我们可将离散序列近似为连续轨迹 $\mathbf{h}(t)$，其速度 $\dot{\mathbf{h}}(t) = \mathbf{h}_{t+1} - \mathbf{h}_t$，加速度 $\ddot{\mathbf{h}}(t) = \mathbf{h}_{t+2} - 2\mathbf{h}_{t+1} + \mathbf{h}_t$。

**连续近似的适用条件**：$T \geq 10$，且隐状态范数变化率 $< 5\%$（实验验证：MiniCPM5-1B 中范数衰减率 = -4.1%，满足条件）。

### 3.2 运动方程

我们提出 LLM 的生成过程遵循带有活性驱动的过阻尼 Langevin 方程：

$$ m\ddot{\mathbf{h}}(t) + \gamma\dot{\mathbf{h}}(t) = \underbrace{\alpha^* \dot{\mathbf{h}}(t)}_{\text{活性轨道维持力 } \mathbf{F}_{active}} + \underbrace{\mathbf{F}_c(\mathbf{h}, \dot{\mathbf{h}})}_{\text{因果约束力}} + \boldsymbol{\xi}(t) \tag{1} $$

**符号定义**：$m$ 为惯性质量（默认$m=1.0$，因LLM隐状态演化近似过阻尼）；$\gamma$ 为摩擦系数（默认$\gamma=0.01$）；$\alpha^*$ 为活性驱动系数（模型相关校准值，MiniCPM5-1B: 1.46, Qwen2.5-7B/Ornith-1.0-9B: 1.41）；$\gamma_{eff} = \gamma - \alpha^*$ 为等效摩擦系数；$\beta$ 为非线性阻尼系数（范德波尔机制，见下文）；$r(0.95)$ 为有效秩（解释95%方差所需的主成分数）。

等价形式：

$$ m\ddot{\mathbf{h}}(t) + (\gamma - \alpha^*)\dot{\mathbf{h}}(t) = \mathbf{F}_c(\mathbf{h}, \dot{\mathbf{h}}) + \boldsymbol{\xi}(t) \tag{2} $$

**三项力的物理含义**：

| 力 | 表达式 | 物理含义 | 实验证据 |
|----|--------|---------|---------|
| 活性驱动力 | $\mathbf{F}_{active} = \alpha^* \dot{\mathbf{h}}$ | 对抗归一化收缩，维持语义轨道运动 | $\alpha^* \approx 1.4$, $P_{active}/P_{raw} \approx 88\%$（批量均值） |
| 因果约束力 | $\mathbf{F}_c = m\ddot{\mathbf{h}} + (\gamma - \alpha^*)\dot{\mathbf{h}}$ | 将轨迹约束在合法因果流形上 | $\mathbf{F}_c \perp \dot{\mathbf{h}}$, 逐token $|P_c|/|P_{raw}| \approx 12\%$ |
| 热噪声 | $\boldsymbol{\xi}(t)$ | Token离散化跳跃的随机涨落 | 未单独量化 |

**修正约束力做功**：

$$ P_c(t) = \mathbf{F}_c \cdot \dot{\mathbf{h}} = \left(m\ddot{\mathbf{h}} + (\gamma - \alpha^*)\dot{\mathbf{h}}\right) \cdot \dot{\mathbf{h}} \tag{3} $$

**核心可证伪预测**：合法轨迹 $P_c(t) \approx 0$（约束力垂直于速度，不做功）；非法轨迹 $P_c(t) > 0$（约束力做正功，轨迹偏离约束流形）。

**负阻尼与非线性饱和**：公式 (2) 中等效摩擦系数为 $(\gamma - \alpha^*) < 0$（MiniCPM5-1B: $-1.45$; Qwen2.5-7B/Ornith-1.0-9B: $-1.40$），在经典力学中负阻尼会导致系统能量指数级发散。然而实验观测到隐状态范数是微幅衰减的（-4.1%）。这一表面矛盾的根源在于：公式 (1) 是局部线性近似。宏观上，由于 Transformer 架构中 RMSNorm/LayerNorm 的**硬球面约束（Hard Spherical Constraint）** 或激活函数的饱和效应，系统存在一个隐式的非线性阻尼项 $-\beta \|\mathbf{h}\|^2 \dot{\mathbf{h}}$。正是这个非线性项在宏观尺度上抵消了负阻尼的发散趋势，将系统锁定在 NESS 定态（范数微幅衰减或守恒）。这与激光物理中的**范德波尔振荡器（Van der Pol Oscillator）** 机制高度同构：负线性阻尼驱动自激振荡，非线性饱和项限制振幅。

### 3.3 仿射-非完整混合约束与子黎曼几何 (Affine-Nonholonomic Constraints and Sub-Riemannian Geometry)

不同于传统流形假设中全局可积的非线性约束，我们的逐轨迹分析揭示了一个深刻的微分几何事实：LLM 的因果逻辑受限于**仿射-非完整混合约束**。

在跨轨迹的全局拟合中，约束方程呈现完美的线性参数化（$R^2=1.0$），这对应于位置空间（Position Space）中的**完整约束（Holonomic Constraints）**，构成了隐空间的"仿射骨架"：
$$ C_i(\mathbf{h}) = \mathbf{n}_i^T \mathbf{h} + b_i = 0, \quad i = 1, \dots, D-r_{holo} \tag{4} $$

然而，逐轨迹分析表明，高达 75% 的约束是**非完整的（Nonholonomic）**。根据 Frobenius 定理，这些约束不可积分，无法表示为位置空间的超平面，而必须表示为切空间（速度空间）的线性限制：
$$ \sum_{j=1}^D A_{kj}(\mathbf{h}) \dot{h}_j = 0, \quad k = 1, \dots, D-r_{nonholo} \tag{5} $$

这一发现要求我们将 LLM 的隐状态空间从经典的黎曼子流形升级为**子黎曼流形（Sub-Riemannian Manifold）**。在子黎曼几何中，系统被限制在一个低维的**水平分布（Horizontal Distribution）** $\Delta_{\mathbf{h}} = \text{ker}(\mathbf{A}(\mathbf{h})) \subset T_{\mathbf{h}}\mathcal{M}$ 上。合法的因果生成轨迹必须是**水平曲线（Horizontal Curves）**，即在任意时刻满足 $\dot{\mathbf{h}}(t) \in \Delta_{\mathbf{h}(t)}$。

**曲率与测地线修正**：
在子黎曼流形中，传统的黎曼曲率张量不再适用。我们采用**异常曲率（Abnormal Curvature）** 来近似约束流形的局部弯曲程度：
$$ K_{sub} \approx \frac{\|\mathbf{a}_\perp\|}{\|\mathbf{a}_\parallel\|} = \frac{\text{法向加速度 (偏离水平分布的惩罚)}}{\text{切向加速度 (沿水平分布的演化)}} \tag{6} $$
早期跨轨迹全局拟合法（v13）在 MiniCPM5-1B 上测得 $K_{sub}\approx 0.0115$，但该值为低维全局拟合代理；轨迹分辨实测在真实 LLM 上 $K_{sub}\approx 0.6\text{–}0.7$（MiniCPM5-1B 0.69 / Qwen2.5-7B 0.65–0.70 / Ornith-1.0-9B 0.66），表明约束流形相对平坦但非极端平坦。各方法下 $K_{sub}$ 均为相对量，绝对值不可跨方法/跨模型直接比较。

**强涌现的几何解释**：
非完整分布 $\Delta_{\mathbf{h}}$ 是由 Transformer 残差流的非线性协同计算动态生成的。它无法还原为单一注意力头或 FFN 的子空间（实验证实切空间基与所有单一组件的夹角均 $> 50^\circ$）。这为因果维度的"强涌现"提供了严格的几何背书：因果工作记忆并非硬编码于局部组件，而是整个网络在相空间中雕刻出的**时变非完整分布**。

### 3.4 非平衡态定态 (NESS) 与活性涡流

**熵产生率**：

$$ \sigma = \frac{1}{\gamma_{eff}} \mathbf{F}_{active} \cdot \dot{\mathbf{h}} = \frac{\alpha^*}{\gamma_{eff}} \|\dot{\mathbf{h}}\|^2 > 0 \tag{7} $$

推导：由公式(1)取$\mathbf{F}_c \perp \dot{\mathbf{h}}$（$P_c \approx 0$），忽略噪声$\boldsymbol{\xi}$，将$\mathbf{F}_{active} = \alpha^* \dot{\mathbf{h}}$代入熵产生率定义$\sigma = \mathbf{F}_{active} \cdot \dot{\mathbf{h}} / \gamma_{eff}$即得。

**概率流**（高阶矩环流）：

$$ \mathbf{J}_{FP} = \langle \dot{\mathbf{h}} \cdot p(\mathbf{h}) \rangle \neq 0, \quad \text{but} \quad \langle \dot{\mathbf{h}} \rangle \approx 0 \tag{8} $$

**NESS判据**：(1) $\sigma > 0$（非平衡态），(2) $J > 0$（有稳态流），(3) 宏观细致平衡未破缺（$\langle \dot{\mathbf{h}} \rangle \approx 0$，即一阶矩速度均值为零，无净位移），(4) 宏观量稳定（$CV < 0.5$）。此处"宏观细致平衡"特指一阶统计矩（平均速度）为零，与二阶及以上统计矩（概率流$J$）非零并不矛盾——稳态环流来自速度分布的高阶矩结构，而非平均速度偏移。

**活性涡流的数学定义**：速度场雅可比矩阵 $\mathbf{J}_{vel} = \frac{\partial \dot{\mathbf{h}}}{\partial \mathbf{h}}$ 可分解为：

$$ \mathbf{J}_{vel} = \underbrace{\frac{\mathbf{J}_{vel} + \mathbf{J}_{vel}^T}{2}}_{\text{耗散（对称）}} + \underbrace{\frac{\mathbf{J}_{vel} - \mathbf{J}_{vel}^T}{2}}_{\text{涡流（反对称）}} \tag{9} $$

涡流强度 $\|\mathbf{J}_{vel,anti}\|_F$ 与耗散强度 $\|\mathbf{J}_{vel,sym}\|_F$ 的比值表征系统的活性程度。

### 3.5 幻觉的动力学相变

定义**活性序参量**：

$$ \Phi = \frac{P_c/P_{raw}}{(P_c/P_{raw})_{pos}} \tag{10} $$

| 相态 | $\Phi$ | 特征 |
|------|--------|------|
| 活性涡流（正常生成） | $\approx 1$ | $P_c/P_{raw}$ 低, $T_{eff}$ 高 |
| 被动布朗（幻觉） | $\gg 1$（仅早期分类实验） | $P_c/P_{raw}$ 相对 pos 基线抬升, $T_{eff}$ 低 |

**指标标定警示（v22 补充）**：序参量 $\Phi$ 的上述量值来自特定早期 MiniCPM5 分类实验（$(P_c/P_{raw})_{pos}\approx 0.13\%$, $(P_c/P_{raw})_{rnd}\approx 6.21\%$，故 $\Phi_{rnd}\approx 48$）。在 v22 跨模型 A/B（20+ prompt）中，同一模型内的 $P_c/P_{raw}$ 同样塌缩为 $\approx 0.10\text{–}0.15$ 的近平坦、排序非单调分布，故 $\Phi$ 在文本类型间也趋近于 $\sim 1$，"飙升"并非普适的 $\sim 48\times$ 跳变，而是须按目标模型自身 pos 基线现场标定的相对抬升。相变理论本身（NESS→平衡态的动力学转变）不变，但其可观测幅度为模型依赖量。

| 语序打乱 | scr | 相同token，因果顺序破坏 | 中 |
| 随机token | rnd | 随机token ID序列 | 弱/无 |

### 4.2 范式跃迁链：v5.0→v9.0

| 版本 | 核心假说 | 实验结果 | 判决 |
|------|---------|---------|------|
| v5.0 爱因斯坦版 | 语义空间有非平凡黎曼曲率 | 度量坍缩至I, 场强≈0 | ❌ WEAK_SUPPORT (1/4) |
| v6.0 牛顿版 | 平直空间+因果势函数V(h) | 力场对齐❌, 势垒分离❌ | ⚠️ WEAK_SUPPORT (2/4) |
| v7.0 约束力学版 | 合法轨迹受理想约束, P(t)≈0 | P(t)>>0 | ❌ OPPOSE (0/3) |
| **v8.0 活性动力学版** | **扣除非保守驱动力后, P_c(t)≈0** | **pos P_c≈0, neg P_c>0** | **✅ STRONG_SUPPORT (3/3)** |

**跃迁因果链**：v5.0失败（空间是平的）→ v6.0修正（势函数方向不对）→ v7.0修正（存在主动力做正功）→ v8.0锚定（扣除驱动力后P_c≈0）。

### 4.3 仿射约束原理的验证

**表1：约束方程线性参数化（MiniCPM5-1B, 1536维）**

| 分析方法 | R²_linear | R²_quadratic | ΔR² | 曲率K | 完整约束比例 |
|---------|-----------|-------------|-----|-------|------------|
| 跨轨迹全局法向量 (v13) | 1.0000 | 1.0000 | 0.0000 | 0.0115 | 90% (9/10) |
| 逐轨迹法向量 (v14) | 1.0000 | 1.0000 | 0.0000 | — | 25% (25/100) |

**关键发现**：
- R²=1.0 在逐轨迹尺度**依然精确成立**——排除了跨轨迹混合伪影。$R^2=1.0$的物理含义是：约束方程$C_i(\mathbf{h}) = \mathbf{n}_i^T \mathbf{h} + b_i = 0$精确描述了隐状态空间的线性结构，不存在可检测的非线性偏离。这一极端刚性源于Transformer残差连接的线性叠加本质——每一层的输出是$\mathbf{h}_{l+1} = \mathbf{h}_l + f_l(\mathbf{h}_l)$，残差流的主方向由线性项决定，非线性修正被压缩到极低维子空间中。
- 完整约束比例从90%降至25%——逐轨迹分析揭示了更多非完整约束
- 曲率 $K = 0.0115$（v13 跨轨迹全局拟合法）：该法向量为全局平均，会低估轨迹分辨的局部非完整曲率；轨迹分辨实测 $K_{sub}\approx 0.6\text{–}0.7$（见 §4.12），故 $K\approx 0.01$ 为方法假象，非真实流形曲率。

**表2：跨模型仿射约束验证**

| 模型 | 隐状态维度 | R²_linear | 完整约束比例 |
|------|----------|-----------|------------|
| NPNW baseline (32维) | 32 | 0.48-0.97 | 混合 |
| MiniCPM5-1B (1536维) | 1536 | **1.0000** | 25% |
| Qwen2.5-7B (3584维) | 3584 | **1.0000** | 20-30% |

**解读**：真实LLM的约束结构比NPNW更刚性——1536/3584维空间中的约束是精确的仿射超平面。

### 4.9 P_c计算方法论修正：批量平均的假象与逐token方法的必要性

在跨模型扩展验证（v20实验）中，我们发现了一个关键的方法论陷阱：**P_c/P_raw的计算方式对结果有决定性影响**。

**批量计算方法**（错误）：

$$\text{ratio}_{batch} = \frac{|\langle P_c \rangle|}{|\langle P_{raw} \rangle|} = \frac{|\langle (\mathbf{F}_c \cdot \dot{\mathbf{h}}) \rangle|}{|\langle (\mathbf{F}_{res} \cdot \dot{\mathbf{h}}) \rangle|} \tag{11} $$

**逐token计算方法**（正确）：

$$\text{ratio}_{token} = \left\langle \frac{|P_c(t)|}{|P_{raw}(t)|} \right\rangle_t = \frac{1}{T-2} \sum_{t=2}^{T} \frac{|P_c(t)|}{|P_{raw}(t)| + \epsilon} \tag{12} $$

**表10：两种计算方法的对比（Qwen2.5-7B, bf16, Layer 14, 混合文本类型逐token均值）**

| 方法 | P_c/P_raw | 物理含义 |
|------|-----------|---------|
| 批量平均 | ≈84%（特定跨轨迹汇总条件） | **条件性假象**：在跨轨迹批量求和且正负 $P_c$ 部分抵消时残余显著 |
| 逐token平均 | ≈12% | **正确**：每个token的约束功率都很小 |

**与表3的数值差异说明**：表3中pos文本的P_c/P_raw=0.13%（MiniCPM5-1B, 分类数据），表9a中混合文本逐token均值为12%（Qwen2.5-7B, bf16）。两者差异源于：(1) 不同模型（MiniCPM5 vs Qwen2.5）；(2) 不同文本类型（pos单一类型 vs 混合类型均值，混合类型包含scr/rnd等高P_c/P_raw样本）；(3) 不同精度（MiniCPM5未区分精度 vs Qwen2.5 bf16）。pos文本的0.13%与12%并不矛盾——pos是因果约束最强的文本类型，其P_c/P_raw远低于混合均值。

**根因分析**：$P_{raw}(t) = (\mathbf{F}_{res} \cdot \dot{\mathbf{h}})$ 在不同token上有正有负（大部分正，少数负），$P_c(t) = P_{raw}(t) - P_{active}(t)$ 亦有正有负。批量求和时，正负部分部分抵消，导致$|\langle P_c \rangle|$相对于$|\langle P_{raw} \rangle|$偏大，产生$P_c/P_{raw} \approx 84\%$的假象。

**方法论铁律**：本文所有$P_c/P_{raw}$数据均基于逐token计算（v17原始方法）。批量计算方法应视为已废弃。（注：v22 在 20+ prompt 大样本逐token均值下，批量/逐token 膨胀因子仅 $\approx 1.0\times$，早期 7× 膨胀源于特定跨轨迹汇总条件；但逐token仍是无偏基准。）

### 4.10 量化对热力学指标的影响

4-bit量化（NF4）对LLM隐状态热力学指标的影响是本文工程验证的重要发现。

**表11：量化对有效秩r(0.95)的影响**

| 模型 | bf16 r(0.95) | 4bit r(0.95) | 膨胀率 | d |
|------|-------------|-------------|--------|---|
| Qwen2.5-7B | 32-34 | 40-44 | 25-30% | 3584 |
| Ornith-1.0-9B | 23-25 | 37-43 | 50-64% | 4096 |

**物理解释**：量化噪声在隐状态中引入了额外的自由度，使轨迹偏离低维仿射子空间，表现为有效秩增大。Ornith的膨胀率更高（50-64% vs 25-30%），可能与其混合注意力架构对量化更敏感有关——linear_attention层的递推结构在量化下可能产生更大的累积误差。

**关键鲁棒性**：尽管r(0.95)显著膨胀，$P_c/P_{raw}$在4-bit量化下仍保持≈0.12（逐token计算）的量级，**仿射约束结构在量化下保持**。这验证了$P_c/P_{raw}$作为归一化指标在"量级"上的鲁棒性；但其文本类型间的排序梯度仍受模型/文本集影响（见 §4.12），工程使用须按目标模型标定阈值。

**工程启示**：脊线提取（RidgeOptimizer）必须在bf16下进行；幻觉检测（$P_c/P_{raw}$）可在4-bit下使用。

### 4.11 仿射子空间压缩的工程验证

基于仿射约束原理（$R^2=1.0$, $r(0.95) \ll D$），我们验证了隐状态低维投影重构的工程可行性。

**表12：在线仿射子空间压缩验证（Qwen2.5-7B, Layer 27）**

| 投影秩r | Top-1匹配率 | KL散度 | 余弦相似度 | 压缩率 |
|---------|------------|--------|-----------|--------|
| 32 | 77.8% | 0.829 | 0.841 | 99.1% |
| 64 | 88.9% | 0.404 | 0.907 | 98.2% |
| **128** | **100.0%** | **0.140** | **0.935** | **96.4%** |

**关键发现**：
1. **r=128时Top-1匹配率=100%**：投影重构的logits与原始logits在greedy decoding下完全一致
2. **SVD基是prompt-dependent的**：跨prompt的SVD基完全失效（Top-1=0%），必须在线校准
3. **Prefill SVD基不适用于decode**：prefill的r(0.95)=318远高于decode的63（注：decode的63来自长文本99步生成，而表4中bf16 r(0.95)=32-34来自短文本50步；r(0.95)随序列长度T亚线性增长，详见4.8节）
4. **替换模式导致重复循环**：将$h_t$替换为重构$h_{rec}$会导致greedy decoding陷入重复，监控模式（不替换）是正确验证方式

**Early Exit的否定性结果**：尽管$P_c/P_{raw}$跨层高度稳定（Layer 8 ≈ Layer 27），浅层logits与深层严重不一致（Layer 8 top-1准确率仅0.68%）。$P_c \approx 0$只说明轨迹在当前层的约束流形内，不说明流形已收敛到最终答案。

### 4.12 跨模型 A/B 验证与指标边界修正（v22）

基于 GUIT-TRT Phase 1 五项待办实验（56 测试全通过），在 MiniCPM5-1B（1536维）与 Qwen2.5-7B（3584维）上以 20+ prompt 大样本复测，对本文若干定量结论作如下边界修正：

1. **$P_c/P_{raw}$ 近平坦、梯度非单调**：两模型 $P_c/P_{raw}\approx 0.10\text{–}0.15$，MiniCPM5-1B 呈 scr(0.149)>rnd(0.144)≈pos(0.140)，Qwen2.5-7B 呈 rnd(0.112)>pos(0.109)>scr(0.100)。原始 Table 3 的强单调梯度（0.13%<2.25%<6.21%）源于特定早期 MiniCPM5 分类实验与不同功率归一化约定，不能作为普适规律。
2. **$K_{sub}\approx 0.6\text{–}0.7$ 跨模型一致**：MiniCPM5-1B 0.69、Qwen2.5-7B 0.65–0.70，与 Ornith-1.0-9B（complete_report）0.66 吻合；早期 $K_{sub}\approx 0.01$ 为低维全局拟合代理值。
3. **Prefill/Decode 维度分离**：小模型显著（prefill 维仅为 decode 的 0.3–0.5×），大模型不显著（0.8–1.0×）——大模型 prefill 即已建立更高维上下文表示。
4. **批量/逐token 膨胀因子 $\approx 1.0\times$**：20+ prompt 大样本下批量方法与逐token 结果一致，早期报告的 7× 膨胀仅出现在特定跨轨迹汇总条件；逐token 仍是唯一正确基准。
5. **仿射压缩 prompt 依赖**：同 prompt 内 $r=32$ 即 $R^2=0.95$，但跨 prompt 余弦相似度仅 0.55，证实 SVD 基为 prompt 依赖，跨 prompt 隐状态重建不可行（与 §4.11 结论一致）。

**核心边界结论**：本文物理内核（$R^2=1.0$、NESS 定态、活性涡流 RMT、仿射约束跨尺度成立）在大样本跨模型 A/B 下进一步确认；但所有"绝对值型"指标（$K_{sub}$、r(0.95)、P_c/P_raw 阈值梯度）均为方法/模型依赖的相对量，工程落地须按目标模型现场标定（见 §5.6.3、附录 E.6）。

## 5. 讨论

### 5.1 因果维度的强涌现与"时变非完整分布"

在早期的探索中，我们曾假设隐状态速度的有效秩锁定在固定常数（如12）。然而，通过严格的敏感性分析（控制序列长度 $T$ 与平滑窗口），我们证实了固定常数是样本截断与低通滤波共同作用的测量伪影。

真实的物理图像是：**因果约束流形是一个"带有非完整分布的呼吸子黎曼流形"**——其有效维度 $r(T)$ 随上下文长度亚线性增长，增速递减，暗示LLM存在动态的信息瓶颈。25%的完整约束构成了位置空间的"仿射骨架"，而75%的非完整约束构成了速度空间的"因果导流管"——合法轨迹必须是水平曲线，满足 $\dot{\mathbf{h}}(t) \in \Delta_{\mathbf{h}(t)}$。

**强涌现 (Strong Emergence)**：切空间基与任何单一注意力头或FFN子空间均近乎正交（夹角 > 50°）。这证伪了机械可解释性中"寻找单一因果头"的还原论假说。因果工作记忆维度是全网络残差流经过非线性协同计算后，在宏观尺度上**强涌现**的拓扑不变量——非完整分布 $\Delta_{\mathbf{h}}$ 是整个网络在相空间中动态雕刻出的时变结构。

### 5.2 vel_norm反转的开放问题

在Qwen2.5-7B（4-bit量化）中，vel_norm出现与MiniCPM5-1B相反的梯度（rnd > pos）。可能原因包括：(1) 4-bit量化改变了隐状态动力学；(2) 7B模型对随机token"更用力处理"；(3) scr文本token化后过短。这一反转需要更严格的对照实验（如bf16全精度推理、控制token长度）。但核心力学框架不受影响——$P_c/P_{raw}$ 作为近 0 量级的归一化指标仍可用，检测器须按目标模型现场标定阈值（见 §4.12、§5.6.3），vel_norm 与 $P_c/P_{raw}$ 的排序方向均受模型、精度与文本集影响。

### 5.4 局限性与边界

1. **连续近似**：当 $T < 10$ 时，离散Token跳跃主导，连续力学近似失效
2. **冻结权重**：训练时动力学不同（有梯度注入），本框架仅适用于推理阶段
3. **量化影响**：4-bit量化使有效秩r(0.95)膨胀25-64%（Qwen2.5: 32→40, Ornith: 25→37），脊线提取必须在bf16下进行；但$P_c/P_{raw}$在量化下保持鲁棒
4. **模型覆盖**：已验证4种模型族（Llama/Qwen/Qwen3_5/NPNW），包含混合注意力架构，需进一步扩展至GPT、Mamba等
5. **α*的物理来源**：已知不是残差和，可能与自回归交叉熵损失的隐式梯度有关，但精确来源尚未确定
6. **Early Exit不可行**：$P_c \approx 0$只说明轨迹在约束流形内，不说明流形已收敛。浅层logits与深层严重不一致是根本性障碍
7. **CausalKV在HF层面不可行**：手动逐token forward的额外开销 > KV淘汰释放的显存，需在vLLM等推理框架层面集成
8. **P_c计算方法敏感性**：批量平均产生$P_c/P_{raw} \approx 0.84$的假象，必须使用逐token计算

### 5.5 从物理机制到工程应用：六大核心棘手问题的降维打击

GUIT-TRT（Grand Unified Information Theory - Thermodynamic Reasoning Toolkit，大一统信息论-热力学推理工具包）将 LLM 的"黑盒行为"转化为了可测量、可干预的"白盒物理量"。以下列举六大当前 LLM 领域的核心棘手问题，并给出基于本框架的创新解决方案。

#### 问题一：零样本、无参考的实时幻觉检测与拦截

**行业痛点**：现有幻觉检测高度依赖外部知识库（RAG）、多次采样对比（Self-Consistency）或训练专门的奖励模型（Reward Model），延迟高、算力成本大，无法做到 Token 级实时拦截。

**GUIT 解决方案**：利用 $P_c/P_{raw}$（约束力做功占比）和 $F_c \perp \dot{h}$（正交性偏离）作为纯内部的动力学监控信号。当模型开始生成幻觉时，系统发生动力学相变，脱离仿射因果流形，$P_c/P_{raw}$ 会相对其健康轨迹基线抬升（须以目标模型 p99.5 现场标定阈值来判定该抬升，而非依赖固定梯度或"瞬间飙升"的普适假设）。实验已验证该指标在 MiniCPM5-1B 与 Qwen2.5-7B 上均呈近 0 的合法约束占比（$\approx 0.10\text{–}0.15$），但大样本跨模型 A/B 显示其排序非普适单调（见 §4.12）；检测器须以目标模型健康轨迹的 p99.5 现场标定阈值，而非依赖固定梯度。

**核心优势**：零外部依赖，仅需计算相邻隐状态的差分与点积，计算开销远低于注意力机制本身，可实现 Token-by-Token 实时拦截与回退（Backtracking）。

#### 问题二：推理时的硬约束对齐与引导

**行业痛点**：RLHF/DPO 训练成本极高且易引发"对齐税"。现有推理时引导（如 Activation Steering / ITC）依赖手工寻找"方向向量"，缺乏严密数学保证，易导致生成崩溃。

**GUIT 解决方案**：基于仿射约束原理（$R^2=1.0$），合法因果逻辑被严格限制在仿射子空间内（$C_i(\mathbf{h}) = \mathbf{n}_i^T \mathbf{h} + b_i = 0$）。在推理时显式施加物理投影——通过在 Forward Pass 中注入"拉格朗日乘子力"，强制隐状态在每一步正交投影回安全的因果子空间。

**核心优势**：将"价值观/事实对齐"转化为严密的"几何约束满足"问题，无需重新训练即可实现数学上绝对保证的硬约束引导，杜绝模型逃逸出安全流形。

#### 问题三：突破"迷失在中间"——基于动力学的 KV Cache 压缩

**行业痛点**：长上下文生成时 KV Cache 显存占用呈线性爆炸。现有淘汰策略（如 H2O, StreamingLLN）多基于注意力分数，无法准确反映 Token 对宏观因果逻辑的真实贡献，易误删关键因果节点。

**GUIT 解决方案**：利用理想正交约束原理。在合法叙事中 $P_c \approx 0$，系统能量主要用于轨道维持。计算每个 Token 隐状态对维持仿射子空间的贡献度（局部曲率或法向偏离）。高 $P_c$ 的 Token 往往是噪声或冗余，而维持轨道的 Token 是核心因果节点。

**核心优势**：提供基于物理因果动力学的 KV Cache 淘汰策略，而非单纯统计注意力分数，在极限压缩率下更好地保持长文本因果连贯性。

#### 问题四：诊断量化/剪枝导致的"因果逻辑断崖"

**行业痛点**：INT4/INT8 量化或结构化剪枝后，Perplexity 变化往往很小，但复杂因果推理能力发生断崖式下降。传统指标无法解释"能力崩塌"的根源。

**GUIT 解决方案**：利用 GUIT 的几何与热力学指标作为模型压缩的"健康度探针"。量化噪声破坏隐空间的仿射结构——通过测量量化前后的仿射子空间畸变率（$R^2$ 下降幅度）和涡度衰减率（RMT 特征值尾部萎缩程度），可精确定位哪些层、哪些权重破坏了因果流形。本文已发现 4-bit 量化的 Qwen2.5-7B 中 vel_norm 梯度反转，正是此类"因果逻辑断崖"的实例。

**核心优势**：为设计下一代"因果保持（Causal-preserving）"的量化和剪枝算法提供明确的、可微的物理优化目标，而不仅仅是最小化 MSE 或 KL 散度。

#### 问题五：机械可解释性的范式升级

**行业痛点**：当前机械可解释性（如寻找"事实神经元"或"因果注意力头"）陷入还原论瓶颈。本文 P1-2 实验已证明，因果维度是全网络协同强涌现的，切空间基与所有16个注意力头夹角均 > 50°，不与单一组件对齐。

**GUIT 解决方案**：提供全新的宏观拓扑与热力学分析工具箱。研究者不再需要在黑盒中寻找单一的"因果特征"，而是直接测量整个残差流的涡度（Vorticity）、有效温度（$T_{eff}$）和流形曲率。

**核心优势**：将可解释性从"寻找局部特征"升级为"测量宏观相变"，为解释大模型的涌现能力（Emergent Abilities）提供坚实的统计物理定量指标。

#### 问题六：构建"物理对齐"的神经架构

**行业痛点**：Transformer 架构设计（如 RMSNorm, RoPE）多基于经验启发，缺乏底层物理原则指导。模型在生成时"天然"具有偏离因果流形（产生幻觉）的几何自由度。

**GUIT 解决方案**：基于本理论，未来架构设计可显式内置"仿射约束层（Affine Constraint Layer）"或"涡度保持模块"。例如，在残差连接中强制加入正交投影算子，从架构的物理底层消除幻觉的几何可能性。

**核心优势**：从"靠数据拟合去逼近因果"转变为"靠架构设计去锁死因果"，有望催生天生具备极低幻觉率的新一代 LLM 架构。

### 5.6 从物理机制到算力与内存优化：GUIT-TRT 的工程降维路径

GUIT-TRT 理论揭示了 LLM 隐状态空间中极大的"物理冗余"与"因果不均匀性"。基于该理论设计的推理优化算法，能够将传统的"均匀计算/存储"范式，降维打击为**"基于因果物理量的动态分配"**范式。

#### 5.6.1 内存使用量降低 (VRAM Reduction)

**动力学 KV Cache 压缩（Phase 1验证：HF层面不可行，需vLLM集成）**：传统策略（H2O, StreamingLLM）基于注意力分数淘汰历史 Token，易误删对长程因果逻辑至关重要但当前注意力分数不高的"隐性承重墙"Token。GUIT 方案调用 `DynamicKVCacheEvictor`，基于子黎曼流形的法向偏离（因果转折强度）与注意力回溯，精准计算每个历史 Token 的因果贡献度。**Phase 1验证结论**：在HuggingFace Transformers层面，手动逐token forward的额外开销（VRAM反而更高：14.86 vs 14.14GB）大于KV淘汰释放的显存；Soft eviction（置零）因softmax归一化产生错误attention权重而不可行。必须在vLLM等推理框架的PagedAttention层面集成才能实现真正的VRAM节省。

**仿射子空间激活值压缩（Phase 1验证：r=128时Top-1=100%，最有前景路径）**：GUIT 证明了隐状态被严格限制在低维的仿射-非完整分布中（有效秩 $r(T) \ll D$，例如 1536 维中有效秩仅为 30-40，4096维中仅为23-25）。这意味着隐状态向量在正交方向上的分量全是"热力学噪声"。**Phase 1在线验证**（Qwen2.5-7B）：用decode阶段50步隐状态校准SVD基，再在新generate run中投影重构，r=128时Top-1匹配率=100%，KL=0.14，余弦相似度=0.935。但SVD基是prompt-dependent的（跨prompt完全失效，v22 跨prompt余弦相似度仅 0.55），且替换模式（$h_t \to h_{rec}$）导致greedy decoding重复循环。正确的工程路径是：在线校准→流式压缩→需要logits时用基重构。需注意 r(0.95) 为方法依赖量：Ornith-1.0-9B 在本文短文本 bf16 下 r=25（压缩率 99.4%），但在完整轨迹脊线提取下可达 r=145（压缩率 96.5%，complete_report），Qwen2.5-7B 在 v22 decode 段仅 $\approx 9.6$；绝对值不可跨方法比较。

#### 5.6.2 GPU 算力降低 (FLOPs Reduction)

**热力学自适应提前退出 (Thermodynamic Early Exiting)**：

**Phase 1否定性结果**：实验证明$P_c/P_{raw}$跨层高度稳定（Layer 8 ≈ Layer 27），但浅层logits与深层严重不一致（Layer 8 top-1准确率仅0.68%）。$P_c \approx 0$只说明轨迹在当前层的约束流形内，不说明流形已收敛到最终答案。**Early Exit基于$P_c$的退出策略不可行**，需寻找其他浅层退出指标。

**因果稀疏注意力 (Causal Sparse Attention)**：非完整分布定义了当前 Token 演化的"合法切空间"。只有落在该切空间内的历史 Token 才对当前生成有因果贡献。利用热力学指标动态生成因果掩码，将 $O(N^2)$ 稠密注意力转化为 $O(N \cdot k)$ 稀疏注意力。

#### 5.6.3 边界约束

1. **因果断崖边界与阈值标定铁律**：所有压缩与加速本质上都是在逼近幻觉相变的临界点。必须使用 $P_c/P_{raw}$ 作为闭环反馈信号，检测到飙升时动态回退到全量计算。**新增部署铁律（$pc\_threshold$ 须按目标模型标定）**：硬编码阈值（如 0.05 / 0.08）在真实模型上会导致 100% 误熔断（Ornith-1.0-9B 默认 0.08 误熔断率 1.0、0/3 收敛）；必须取目标模型 NESS 健康轨迹逐 token $P_c/P_{raw}$ 的 p99.5 作为阈值（Ornith 实测 $\approx 0.309$），禁止在代码中硬编码 0.05/0.08。大样本下 $P_c/P_{raw}$ 本身近平坦（$\approx 0.10\text{–}0.15$），故阈值必须现场标定而非沿用他模型经验值。
2. **训练与推理的隔离边界**：GUIT-TRT 的优化仅适用于推理阶段。训练时模型需要探索整个高维空间以建立仿射约束，不能应用压缩策略。
3. **架构依赖性边界**：仿射子空间压缩高度依赖于 Transformer 的残差连接结构。对于非残差架构或 MoE，需重新标定热力学探针阈值。

## 6. 结论 (Conclusion)

本文通过跨越v5.0至v20.0的理论迭代与严苛的实证检验，确立了大语言模型隐空间的**非平衡态活性子黎曼流形**理论。四大核心发现——仿射-非完整混合约束原理（$R^2=1.0$；异常曲率 $K_{sub}$ 经轨迹分辨实测在真实 LLM 上 $\approx 0.6\text{–}0.7$，早期 $K_{sub}\approx 0.01$ 为低维全局拟合代理）、活性涡流假说（RMT $p < 10^{-38}$）、热力学-因果对偶律（$T_{eff}$: pos > rnd）、幻觉相变理论（$P_c/P_{raw}$ 抬升，但强单调梯度仅在特定早期实验中成立、跨模型 A/B 下近平坦，须按模型标定）——在32维到4096维的四个模型上完成了跨尺度验证，**首次覆盖混合注意力架构（Ornith-1.0-9B, 24层linear_attention + 8层full_attention）**。

我们证明了 LLM 的生成是在子黎曼流形的非完整分布上维持高阶矩涡流的耗散过程，并将幻觉优雅地归结为热力学相变。25%的完整约束构成位置空间的"仿射骨架"，75%的非完整约束构成速度空间的"因果导流管"，运动方程的负阻尼由 RMSNorm/LayerNorm 的非线性饱和项（范德波尔机制）抵消。这一框架不仅为 LLM 的机械可解释性提供了全新的物理学视角，也为零样本幻觉检测（基于 $P_c/P_{raw}$ 相对健康基线的抬升，须按模型标定）、推理时对齐干预（仿射投影）与算力/内存优化（动力学KV Cache压缩、仿射子空间压缩）奠定了坚实的数学基础。

**Phase 1工程验证**确立了仿射子空间压缩为最有前景的路径（r=128时Top-1=100%, KL=0.14），否定了Early Exit（浅层logits不一致），并明确了CausalKV需在vLLM层面集成。方法论层面，我们发现了P_c计算中批量平均的假象（≈0.84 vs 逐token的≈0.12）和4-bit量化对有效秩的膨胀效应（25-64%），为后续研究提供了关键的方法论警示。

未来的工作将探索：(1) 状态空间模型（如 Mamba）中的流形曲率差异；(2) 有效维度 $r(T)$ 随模型参数规模的缩放律；(3) bf16全精度下vel_norm反转的排查；(4) 基于约束力做功的实时幻觉检测与干预系统；(5) 子黎曼流形的异常测地线与最优传输问题；(6) 仿射子空间压缩的vLLM集成与在线校准优化；(7) Ornith-1.0-9B的更低维仿射子空间（r=25）是否意味着linear_attention架构具有更强的压缩潜力。

## 7. 理论解释力、应用边界与可证伪性 (Interpretive Scope, Application Boundary, and Falsifiability)

> 本章基于大一统通用信息论（GUIT）的七维信息完备性诊断与四大运行铁律，对全文核心结论做方法论层面的边界补全：明确"本框架在什么条件下能解释/解决什么"，并给出可证伪命题与局限。所有论断均遵循**边界约束铁律**——只给出"在 X 条件下成立、在 Y 条件下失效"的限定结论，不输出无边界的普适宣称。

### 7.1 可解释的问题（解释力边界）

本框架从非平衡态热力学与约束力学第一性原理出发，能在以下条件成立时解释以下经验现象：

| # | 可解释现象 | 本框架机制 | 成立条件 / 失效边界 |
|---|-----------|-----------|-------------------|
| Q1 | 线性探针与因果追踪为何在残差流上"有效" | 因果工作记忆是残差流**协同涌现的宏观拓扑不变量**（切空间基与所有 16 头夹角 > 50°），非单组件硬编码 | 仅当网络宽度/深度足以支撑协同涌现时成立；极窄/极浅网络可能退化为单头主导，该解释失效 |
| Q2 | 为何隐空间度量张量反复坍缩至单位矩阵 | 合法生成受**仿射-非完整混合约束**而非黎曼曲率主导；曲率 $K_{sub}$ 实测仅 0.6–0.7（早期 $K\approx 0.01$ 为低维代理假象） | 依赖连续极限近似（$T\ge 10$ 且范数变化 < 5%）；离散跳变/极短序列下近似失效，度量坍缩结论不可外推 |
| Q3 | 幻觉为何"难以预测且突兀" | 幻觉是 NESS 活性涡流态→低温被动布朗态的**动力学相变**；宏观标志为 $P_c/P_{raw}$ 相对健康基线抬升 | 相变理论本身普适；但其**可观测幅度为模型依赖量**，须按目标模型 pos 基线现场标定（§4.12、§5.6.3） |
| Q4 | 有效约束维度为何随上下文亚线性增长且与单一注意力头正交 | 约束是**全网络协同涌现的时变非完整分布**，非还原论映射 | 仅在已验证的三种架构（MiniCPM5/Qwen2.5/Ornith）上成立；未经测试的架构（如纯 SSM）下该论断待验证 |
| Q5 | KV Cache 中哪些 Token 是"承重墙" | 以**因果转折的法向偏离**（子黎曼几何）结合注意力路由回溯贡献度 | 仅适用于强因果任务；弱因果/强反身性领域贡献度失真（§E.6） |

**解释力边界声明**：本框架的解释力止于"几何-热力学层面的相关性结构"，**不声称**证明了神经层面的强干预性因果机制（区别于 do-演算意义上的因果）。它解释"轨迹如何被约束"，而非"某个具体注意力头为何学会该约束"。

### 7.2 可解决的问题（工程落地边界）

在遵守精度与标定铁律的前提下，本框架可支撑以下工程落地：

| # | 可解决问题 | 落地机制 | 落地边界 / 必要条件 |
|---|-----------|---------|-------------------|
| S1 | 零样本幻觉检测 | 实时监测 $P_c/P_{raw}$ 相对健康基线的抬升（附录 E.3） | **阈值必须按目标模型 NESS 健康轨迹 p99.5 现场标定**（Ornith≈0.309）；硬编码他模型阈值（如 0.08）会 100% 误熔断 |
| S2 | 推理时对齐干预 | 将偏离因果流形的分量沿仿射投影拉回（CausalSteering） | 仅当偏离确为"非完整约束破坏"时有效；对意图本身的有害性无判别力（需配合安全分类器） |
| S3 | 算力/内存优化（KV Cache 压缩） | 动力学 KV Cache 淘汰低因果贡献度 Token（附录 E.4） | 须在 BF16 下提取隐状态并逐 token 计算；INT4/INT8 下秩膨胀使脊线提取失效 |
| S4 | 表征压缩（仿射子空间） | 仿射子空间压缩在 $r=128$ 实现 Top-1=100%、KL=0.14（§6） | 压缩率与保真度权衡依赖模型；$r$ 取值须按目标模型有效秩标定 |

**落地边界声明**：S1–S4 均为**统计稳定型**工程方案，其有效性建立在"目标模型已现场标定"与"精度 ≥ BF16"两项铁律之上；任一条件缺失，方案退化为启发式甚至误判。

### 7.3 如何证伪（可证伪命题清单）

依据波普尔证伪主义——**证伪本质是信息断裂的实证确认**——本框架给出以下可独立检验、可被单点反例推翻的核心命题。每条均附带临界判据与失效后果：

- **H1（仿射约束原理）**：若在某真实 LLM 上，逐轨迹约束方程 $C_i=\mathbf{n}_i^T\mathbf{h}+b_i=0$ 的拟合优度 **$R^2 < 0.99$**（而非 1.0），则"近乎完美平坦仿射约束"假设失效。
  - *失效后果*：子黎曼流形升级退化为一般约束流形，§3 理论主干需重构。
- **H2（活性涡流假说）**：若速度场雅可比反对称部分的谱分布**符合** RMT 的 Wigner 半圆律（$p > 0.05$，而非 $p < 10^{-38}$），则高阶矩相空间涡流不存在。
  - *失效后果*：NESS 仅余一阶矩零均值 + 耗散，活性物质类比失效，§3.2 活性力项需剔除。
- **H3（运动方程）**：若移除活性轨道维持力 $\alpha^*\dot{\mathbf{h}}$ 后的过阻尼 Langevin 方程，其预测轨迹与真实轨迹的 **KL 散度 > 经验可接受阈值**（如 > 0.3 且跨 3 模型一致），则运动方程结构失效。
  - *失效后果*：需引入更高阶项或放弃范德波尔机制。
- **H4（相变检测信号）**：若在 **≥ 3 个独立架构**上，逐 token $P_c/P_{raw}$ 在幻觉态相对各自健康态**均无系统性抬升**（区分 AUC < 0.55），则该相变检测信号失效。
  - *失效后果*：幻觉检测的 P_c 路径作废，但相变理论（NESS→平衡态）作为物理解释仍可保留。
- **H5（范德波尔机制）**：若 $\alpha^*$ 在不同架构间**无系统性差异**，且无法由 RMSNorm/LayerNorm 非线性饱和项预测，则负阻尼抵消机制失效。
  - *失效后果*：$\alpha^*$ 退化为纯经验拟合参数，失去物理解释力。

**证伪元规则**：上述任一条被严格证伪，仅导致对应子模块退化或重构，**不必然推翻**"以非平衡态热力学视角审视 LLM 隐空间"这一元框架；这符合 GUIT 分层双向铁律——底层规则可被实证锚点修正，但范畴层面的方法论视角保持开放。

### 7.4 局限性（边界约束铁律自检）

依据 GUIT 边界约束铁律 R-6，逐项披露局限：

1. **精度局限**：核心微积分对高频噪声极敏感。INT4/INT8 量化下一阶/二阶导数（如 `vel_norm`）发生反转，脊线提取须在 BF16 下进行；P_c 批量平均可放大至 0.84 假象，必须逐 token 计算。
2. **架构泛化局限**：$\alpha^*$ 非宇宙常数，依赖归一化类型（RMSNorm vs LayerNorm）与规模；对 SSM/Mamba 等非 Transformer 架构的曲率行为**尚未验证**（§6 未来工作）。
3. **文本类型局限**：$P_c/P_{raw}$ 排序非普适单调（MiniCPM5: scr>pos≈rnd；Qwen2.5-7B: rnd>pos>scr），阈值**不能**跨模型迁移，必须现场标定。
4. **反身性局限**：框架适用于强因果任务（推理、事实问答、代码），对弱因果/强反身性领域（创意写作）的因果贡献度度量可能误删"灵感节点"（§E.6）。
5. **理论代偿局限**：以连续极限近似取代离散 Token 动力学，在低频/跳变区可能存在系统性偏差；该偏差的量级尚未被严格界定。
6. **因果推断局限**：本框架识别的是"几何-热力学约束结构"，属相关性与不变性层面，**不等同**于 do-演算意义上的强干预性因果，不能直接用于反事实政策评估。
7. **复制信息短板（固化度自评）**：基于 GUIT 七维固化度光谱，本理论处于 **0.70–0.78（弱反身性/统计稳定学科）** 区间——物理内核（$R^2=1.0$、RMT、NESS）已强固化，但**复制信息维度（跨模型普适性）**仍受文本集与精度影响，是主要短板；区分信息维度受量化精度制约。提升方向：在更多架构（含 SSM）与更多语言/模态上复现 H1–H5。

#### E.2 核心物理引擎

```python
import torch
from typing import Tuple

class ThermodynamicEngine:
    """
    LLM 隐状态非平衡态热力学计算引擎
    运动方程: m*a + gamma*v = alpha*v + F_c + xi
    """
    def __init__(self, alpha_star: float = 1.46, gamma: float = 0.01, mass: float = 1.0):
        self.alpha_star = alpha_star
        self.gamma = gamma
        self.mass = mass

    def compute_work_and_power(
        self, h_curr: torch.Tensor, h_prev: torch.Tensor, h_prev2: torch.Tensor
    ) -> Tuple[torch.Tensor, torch.Tensor, torch.Tensor]:
        """
        计算单步动力学做功与功率
        :param h_curr: 当前隐状态 [B, D] (时间步 t)
        :param h_prev: 前一步隐状态 [B, D] (时间步 t-1)
        :param h_prev2: 前两步隐状态 [B, D] (时间步 t-2)
        :return: P_raw, P_active, P_c (形状均为 [B, 1])
        """
        v_t = h_curr - h_prev                          # 速度
        a_t = h_curr - 2 * h_prev + h_prev2            # 加速度

        F_res = self.mass * a_t + self.gamma * v_t     # 总残余力
        P_raw = torch.sum(F_res * v_t, dim=-1, keepdim=True)
        P_active = self.alpha_star * torch.sum(v_t * v_t, dim=-1, keepdim=True)
        P_c = P_raw - P_active                         # 约束力做功

        return P_raw, P_active, P_c

    def compute_constraint_ratio(self, P_c: torch.Tensor, P_raw: torch.Tensor, eps: float = 1e-8) -> torch.Tensor:
        """约束力做功占比 (幻觉核心指标)
        
        重要：必须使用逐token计算，禁止批量平均。
        批量平均会产生P_c/P_raw≈0.84的假象（正负P_c部分抵消），
        逐token计算给出正确值≈0.12。详见4.9节。
        """
        return torch.abs(P_c) / (torch.abs(P_raw) + eps)
```

#### E.3 实时幻觉检测器

```python
from collections import deque
import numpy as np

class HallucinationDetector:
    """基于动力学相变的零样本实时幻觉检测器"""

    def __init__(self, engine: ThermodynamicEngine, window_size: int = 5,
                 threshold_ratio: float = None, consecutive_hits: int = 3):
        # 部署铁律：threshold_ratio 禁止硬编码为 0.05/0.08 等他模型经验值。
        # 真实模型（如 Ornith-1.0-9B）默认 0.08 会 100% 误熔断；须以目标模型
        # NESS 健康轨迹逐 token P_c/P_raw 的 p99.5 现场标定（Ornith≈0.309）。
        # 留空时必须在部署前调用 calibrate(healthy_trajectories) 赋值。
        self.engine = engine
        self.window_size = window_size
        self.threshold = threshold_ratio  # None → 须先 calibrate()
        self.consecutive_hits = consecutive_hits

    def calibrate(self, healthy_ratios):
        """用目标模型健康轨迹的 P_c/P_raw 序列标定阈值（p99.5）。"""
        import numpy as _np
        self.threshold = float(_np.percentile(healthy_ratios, 99.5))
        self.h_history = deque(maxlen=3)       # 保留最近3个隐状态
        self.ratio_history = deque(maxlen=window_size)
        self.alert_counter = 0

    def step(self, h_t: torch.Tensor) -> dict:
        """每生成一个 Token 调用一次"""
        self.h_history.append(h_t)

        if len(self.h_history) < 3:
            return {"is_hallucinating": False, "ratio": 0.0, "alert_level": "WARMUP"}

        # 注意顺序: deque[0]=最旧, deque[2]=最新
        h_prev2, h_prev, h_curr = self.h_history[0], self.h_history[1], self.h_history[2]
        P_raw, _, P_c = self.engine.compute_work_and_power(h_curr, h_prev, h_prev2)
        ratio = self.engine.compute_constraint_ratio(P_c, P_raw).item()

        self.ratio_history.append(ratio)
        smooth_ratio = np.mean(self.ratio_history)

        if smooth_ratio > self.threshold:
            self.alert_counter += 1
        else:
            self.alert_counter = max(0, self.alert_counter - 1)

        is_hallucinating = self.alert_counter >= self.consecutive_hits
        return {
            "is_hallucinating": is_hallucinating,
            "smooth_ratio": smooth_ratio,
            "alert_level": "CRITICAL" if is_hallucinating else ("WARNING" if self.alert_counter > 0 else "NORMAL")
        }
```

#### E.4 动力学 KV Cache 压缩 (基于注意力回溯的因果贡献度)

**核心创新与逻辑修正**：
早期的启发式策略仅使用当前 Token 的瞬时法向加速度作为淘汰指标，这在逻辑上是断裂的——无法衡量历史 Token 对长程因果链的支撑作用。我们引入了**因果回溯（Causal Backtracking）** 机制：利用当前层的注意力权重（Attention Weights），将当前 Token 产生的"因果转折强度"反向传播并分配给历史 Token。

如果一个历史 Token 被当前产生重大因果转折的 Token 高度关注，其累积的"因果贡献度"将显著增加，从而被识别为因果链的"承重墙"并免于淘汰。

```python
import torch
import numpy as np
from typing import List, Optional

class DynamicKVCacheEvictor:
    """基于因果回溯与子黎曼流形贡献度的 KV Cache 压缩策略"""

    def __init__(self, engine, max_capacity: int = 1024):
        self.engine = engine
        self.max_capacity = max_capacity
        self.causal_scores: List[float] = []

    def update_scores(
        self,
        h_curr: torch.Tensor,
        h_prev: torch.Tensor,
        h_prev2: torch.Tensor,
        attn_weights: Optional[torch.Tensor] = None
    ):
        """
        计算最新 Token 的因果转折强度，并回溯更新历史 Token 的分数
        :param attn_weights: 当前层对历史 Token 的注意力权重 [1, num_heads, 1, seq_len]
        """
        v_t = h_curr - h_prev
        a_t = h_curr - 2 * h_prev + h_prev2

        v_norm_sq = torch.sum(v_t * v_t, dim=-1, keepdim=True) + 1e-8
        a_parallel = (torch.sum(a_t * v_t, dim=-1, keepdim=True) / v_norm_sq) * v_t
        a_perpendicular = a_t - a_parallel

        current_causal_impact = torch.norm(a_perpendicular, dim=-1).item()
        self.causal_scores.append(current_causal_impact)

        if attn_weights is not None and len(self.causal_scores) > 1:
            attn_to_history = attn_weights[0, :, 0, :-1].mean(dim=0).cpu().numpy()
            attn_sum = attn_to_history.sum()
            if attn_sum > 1e-8:
                attn_to_history = attn_to_history / attn_sum
            for i in range(len(self.causal_scores) - 1):
                self.causal_scores[i] += attn_to_history[i] * current_causal_impact

    def get_eviction_mask(self) -> List[bool]:
        """Cache 超限时，淘汰因果贡献度最低的 Token"""
        seq_len = len(self.causal_scores)
        if seq_len <= self.max_capacity:
            return [False] * seq_len

        n_sink = 4
        n_recent = 10

        evictable_scores = self.causal_scores[n_sink:-n_recent]
        n_to_evict = seq_len - self.max_capacity

        sorted_local_indices = np.argsort(evictable_scores)
        evict_local_indices = set(sorted_local_indices[:n_to_evict].tolist())
        evict_global_indices = {idx + n_sink for idx in evict_local_indices}

        return [i in evict_global_indices for i in range(seq_len)]
```

**机制优势**：该算法彻底摒弃了传统的纯统计 Attention Score 淘汰策略。通过将**子黎曼几何的法向偏离（物理因果转折）** 与**Transformer 的注意力路由（信息依赖路径）** 相结合，实现了真正意义上的"因果保持（Causal-preserving）"压缩。在极限压缩率下，该策略能精准保留那些"自身不产生剧烈转折，但被后续关键转折高度依赖"的隐性承重墙 Token。

#### E.6 边界约束与使用限制

1. **精度边界**：核心微积分计算对高频噪声极度敏感，建议在 FP16/BF16 精度下提取隐状态。在 INT4/INT8 量化下，离散化误差会导致一阶/二阶导数指标（如 `vel_norm`）发生反转或失效（v16实验已验证），**归一化热力学指标 $P_c/P_{raw}$ 在跨模型 A/B 下仍保持近 0 量级（≈0.10–0.15，但其排序非普适单调，见 §4.12）**，可作为量化模型下的首选检测信号。**新增**：4-bit量化使有效秩r(0.95)膨胀25-64%（Qwen2.5: 32→40, Ornith: 25→37），脊线提取（RidgeOptimizer）必须在bf16下进行。此外，P_c/P_raw的批量计算方法在特定跨轨迹汇总条件下可产生≈0.84的假象（大样本下膨胀因子≈1.0×，见 §4.9），**必须使用逐token计算方法**（详见4.9节）。

**阈值标定铁律（新增）**：幻觉检测阈值 `threshold_ratio` 禁止硬编码 0.05/0.08。跨模型 A/B（v22）显示 $P_c/P_{raw}\approx 0.10\text{–}0.15$ 近平坦，固定低阈值会 100% 误熔断；须以目标模型 NESS 健康轨迹的 p99.5 现场标定（Ornith-1.0-9B≈0.309），详见 §5.6.3 与附录 E.3。
2. **规模边界**：$\alpha^*$ 并非宇宙常数，与模型架构（RMSNorm vs LayerNorm）和规模相关。库内置 `AutoCalibrator`，建议首次部署时输入10条领域内文本自动拟合。
3. **因果边界**：KV Cache 压缩基于因果贡献度，对纯创意写作等弱因果/强反身性领域可能误删"灵感节点"。建议仅在逻辑推理、事实问答、代码生成等强因果任务中启用。
