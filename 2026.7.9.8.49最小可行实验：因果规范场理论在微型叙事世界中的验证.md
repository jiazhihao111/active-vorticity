## 最小可行实验：因果规范场理论在微型叙事世界中的验证

基于六顶帽讨论的结论，我们设计一套可直接实现的最小可行实验方案，以低成本检验理论核心主张是否成立。

---

### 一、实验设计原则

| 原则 | 说明 |
|------|------|
| **最小规模** | 模型参数量控制在 10M 以内，可在单张消费级 GPU 上完成全部训练 |
| **完全可控** | 使用程序化生成的合成数据，因果关系已知且可精确标注 |
| **清晰判决** | 每个实验都有明确的“支持/反对”判决标准 |
| **模块化** | 各实验独立可复现，可逐步递进验证 |

---

### 二、实验环境构建：“微型物理叙事世界”

#### 2.1 世界规则定义

构造一个极度简化的叙事环境，称为 **“微型物理叙事世界” (Nano-Physical Narrative World, NPNW)**：

**物理因果规则（显式、可枚举）**：
```
实体类型: {角色, 物品, 门, 开关, 光源}
角色状态: {位置(x,y), 持有物品, 体力值}
动作集合: {移动(方向), 拾取(物品), 使用(物品,目标), 推(门), 等待}
物理规则:
  - 移动: 位置+1，体力-1；目标格不可为墙壁
  - 拾取: 物品必须在同位置且体力>0
  - 使用钥匙开门: 必须持有钥匙，门必须在相邻格
  - 体力=0时无法执行任何动作（只能等待恢复）
```

**叙事因果规则（可编程）**：
```
叙事模板: “获取钥匙 → 开门 → 取得宝藏”
伏笔结构: 钥匙必须在取得前被“提及/看见”至少一次
闭合规则: 故事必须以“取得宝藏”或“失败”结尾
```

**心理因果规则（简化为角色约束）**：
```
角色人格模板: {勇敢型, 谨慎型, 贪婪型}
勇敢型: 体力>0时优先前进，不会等待
谨慎型: 体力<3时优先等待恢复
贪婪型: 看到物品必定拾取，即使会耗尽体力
```

#### 2.2 数据生成

程序化生成 10,000 个故事，每个故事包含 10-30 个叙事步骤。

**数据结构示例**：
```
故事ID: 001
步骤1: [状态] 角色在(0,0)，体力=5，看见前方有门
步骤2: [动作] 移动到(1,0) [物理因果: ✅ 位置连续]
步骤3: [状态] 角色在(1,0)，体力=4，看见钥匙在(2,1)
步骤4: [动作] 移动到(2,1) [叙事因果: ✅ 钥匙被提及]
步骤5: [动作] 拾取钥匙 [物理因果: ✅ 同位置，体力>0]
...
步骤N: [动作] 使用钥匙开门 [叙事因果: ✅ 伏笔回收]
步骤N+1: [状态] 取得宝藏 [叙事因果: ✅ 闭环完成]
```

**正负例构造**：
- **正例**：程序生成的合法状态转移，遵循所有三层因果规则
- **负例**（关键）：通过对正例进行受控扰动生成
  - 物理负例：动作违反物理规则（如穿墙移动、凭空拾取）
  - 叙事负例：打破伏笔-回收链（如使用未被提及的钥匙）
  - 心理负例：角色行为违背人格模板（如勇敢型无理由等待）

### 三、实验一：因果曲率与生成质量的相关性

**目的**：验证理论的**核心预测1**——因果曲率代理量与逻辑一致性评分呈负相关。

#### 3.1 模型

训练一个微型 Transformer（2层，d_model=128，参数量约 2M）作为基线模型，仅在 NPNW 合法故事上做标准下一 token 预测训练。

#### 3.2 因果曲率代理量 \(\tilde{F}_t\) 的构造

在未引入显式规范场的情况下，用以下可计算方法作为场强的代理：

**方法A：隐状态转移不一致分数**
1. 在合法数据上训练一个二分类器 \(C_{\text{legal}}(\mathbf{h}_t, \mathbf{h}_{t+1})\)，判断转移是否因果合法。
2. 在推理时，对每一步转移，计算：
   \[
   \tilde{F}_t = 1 - C_{\text{legal}}(\mathbf{h}_t, \mathbf{h}_{t+1})
   \]
   即分类器判定该转移为“非法”的概率。

**方法B：注意力几何离散度**
1. 提取每一步的注意力权重矩阵 \(A_t \in \mathbb{R}^{n \times n}\)。
2. 合法转移中，注意力应集中在因果相关的 token 对上（如“钥匙”与“开门”）。
3. 定义：
   \[
   \tilde{F}_t = \text{JS}(A_t^{\text{actual}} \| A_t^{\text{expected}})
   \]
   其中 \(A_t^{\text{expected}}\) 是基于因果图谱的先验注意力分布，JS 为 Jensen-Shannon 散度。

#### 3.3 测试与指标

用训练好的模型生成 500 个新故事（给定起始状态，自回归生成）。

对每个故事：
- 自动标注每步动作是否物理合法、是否回收前文伏笔、是否符合角色人格
- 计算每步的 \(\tilde{F}_t\)（使用上述方法A/B）
- 人工或规则自动评分：每步逻辑一致性（0/1）

**统计检验**：
- 计算 \(\tilde{F}_t\) 与逻辑一致性评分的点双列相关系数
- 计算在逻辑断裂点（前后两步）的 \(\tilde{F}_t\) 是否显著高于连贯点（t检验）
- 可视化：沿故事步数绘制 \(\tilde{F}_t\) 曲线，标注断裂点

**判决标准**：
| 结果 | 结论 |
|------|------|
| \(\tilde{F}_t\) 与逻辑分相关系数 r > 0.3，且断裂点 \(\tilde{F}_t\) 显著更高 (p<0.01) | **支持**理论核心主张 |
| 无显著相关性 (r < 0.1) | **削弱**理论，需重新审视因果曲率概念 |
| 负相关 (r < -0.2) | **强反对**，理论方向可能错误 |

### 四、实验二：因果几何损失对长程一致性的影响

**目的**：验证理论的**核心预测2**——加入因果几何损失能提升长程生成一致性。

#### 4.1 对比训练设置

| 组别 | 模型 | 训练损失 | 说明 |
|------|------|----------|------|
| **基线组** | Transformer (2M) | \(\mathcal{L}_{\text{LM}}\) | 标准下一token交叉熵 |
| **实验组** | Transformer (2M) | \(\mathcal{L}_{\text{LM}} + \lambda \mathcal{L}_{\text{gauge}}\) | 加入因果几何正则化 |

#### 4.2 因果几何损失 \(\mathcal{L}_{\text{gauge}}\) 的实现

**简化方案（不实现完整规范场，而是直接塑造表征几何）**：

\[
\mathcal{L}_{\text{gauge}} = \frac{1}{N} \sum_{i=1}^{N} \left[ d(\mathbf{h}_t^i, \mathbf{h}_{t+1}^{i,\text{pos}}) - d(\mathbf{h}_t^i, \mathbf{h}_{t+1}^{i,\text{neg}}) + m \right]_+
\]

其中：
- \(d(\cdot, \cdot)\) 为隐状态间的余弦距离或欧氏距离
- \((\mathbf{h}_t, \mathbf{h}_{t+1}^{\text{pos}})\) 为合法转移对（正例）
- \((\mathbf{h}_t, \mathbf{h}_{t+1}^{\text{neg}})\) 为非法转移对（负例）
- \(m\) 为边际超参数（如 1.0）
- \([\cdot]_+\) 为 ReLU 截断

**直观意义**：迫使合法转移在表征空间中的距离至少比非法转移小 \(m\)，从而在几何上区分因果对错。

#### 4.3 评估设计

- 训练两组模型至收敛（验证困惑度稳定）
- 对每组模型，生成不同长度的故事：**短程**（5-10步）、**中程**（15-20步）、**长程**（25-30步）
- 每种长度生成 200 个故事

**评估指标**：
1. **物理合法率**：自动检查每步动作是否违反物理规则
2. **叙事闭合率**：故事是否以合法闭环结尾
3. **人格一致性**：角色行为是否符合其人格模板的期望
4. **困惑度（PPL）**：作为语言流畅性的对照指标

**判决标准**：
| 结果 | 结论 |
|------|------|
| 实验组长程一致性指标显著高于基线组 (p<0.05)，且短程PPL无显著退化 | **强烈支持**理论 |
| 实验组长程指标略高但未达显著，或PPL严重退化 | **弱支持**，需调整 \(\lambda\) |
| 实验组所有指标均未优于基线组 | **反对**因果几何损失的有效性 |
| 实验组所有指标均差于基线组 | **强反对**，理论核心训练主张有误 |

### 五、实验三：有效记忆核数量与生成收敛性

**目的**：验证理论的**核心预测3**——更多激活的记忆核导致更低路径发散度。

#### 5.1 记忆核的实现

对训练数据中的所有状态转移对 \((\mathbf{h}_t, \mathbf{h}_{t+1})\) 进行 k-means 聚类，得到 \(K\) 个聚类中心作为“记忆锚点” \(\{\mathbf{m}_k\}_{k=1}^K\)。每个簇内的转移模式（平均转移向量）作为该记忆核的“局部规范连接” \(\mathbf{A}_k\)。

#### 5.2 实验设计

- 使用实验二中的因果正则化模型作为基础模型
- 在推理时，根据当前状态 \(\mathbf{h}_t\) 计算与各记忆锚点的相似度
- 通过**阈值控制**激活的记忆核数量 \(K_{\text{eff}}\)：
  - 仅保留相似度前 \(K_{\text{eff}}\) 的记忆核
  - 使用它们的加权平均来修正 logits

**独立变量**：\(K_{\text{eff}} \in \{1, 2, 4, 8, 16, 32, 64, \text{all}\}\)

**测试**：对每个 \(K_{\text{eff}}\) 值，给定同一前缀，生成 100 条不同的完整故事。

**评估指标**：
1. **路径发散度**：100条路径的隐状态序列两两之间的平均弗雷歇距离
2. **因果一致性得分**：同实验二的自动评分
3. **生成多样性**：100条故事中不同结局的数量（控制指标，确保不是简单崩溃）

**判决标准**：
| 结果 | 结论 |
|------|------|
| 路径发散度随 \(K_{\text{eff}}\) 单调递减，且一致性得分单调递增 | **支持**记忆-收敛定理 |
| 两者均不随 \(K_{\text{eff}}\) 变化 | **反对**记忆核机制 |
| 多样性随 \(K_{\text{eff}}\) 急剧下降（所有路径相同） | 收敛过度，需调整机制 |

### 六、实验四：规范场平坦化的可解释性

**目的**：验证理论的**核心预测4**——因果正则化模型的隐状态轨迹更“平坦”。

#### 6.1 方法

1. 取实验二中**基线模型**和**因果正则化模型**各自生成的故事
2. 对每个故事，计算沿生成路径的**连续隐状态转移向量的曲率**：
   \[
   \kappa_t = \frac{\|\Delta\mathbf{h}_t - \Delta\mathbf{h}_{t-1}\|}{\|\Delta\mathbf{h}_t\| \cdot \|\Delta\mathbf{h}_{t-1}\|}
   \]
   其中 \(\Delta\mathbf{h}_t = \mathbf{h}_{t+1} - \mathbf{h}_t\)。
   这是离散曲率的简单近似，衡量转移方向的突变程度。

3. 计算每条路径的平均曲率 \(\langle \kappa \rangle\) 和最大曲率 \(\kappa_{\text{max}}\)

#### 6.2 判决标准

| 结果 | 结论 |
|------|------|
| 因果正则化模型生成路径的 \(\langle \kappa \rangle\) 和 \(\kappa_{\text{max}}\) 均显著低于基线模型 | **支持**“场平坦化”主张 |
| 两者无显著差异 | **反对**几何解释的实质有效性 |
| 因果正则化模型曲率更高 | **强反对** |
