    *   *信息失真*：当LLM生成文本时，如果它没有沿着语义流形的测地线演化，而是被噪声或错误的Prompt“拉扯”偏离了测地线，就会产生额外的“能量代价”（表现为困惑度Perplexity激增）。**幻觉，本质上是生成路径偏离了因果测地线。**
2.  **和乐异常（Holonomy Anomaly / 相位不守恒）**：
    *   *几何意义*：在弯曲流形上，一个向量沿着闭合回路平行移动一圈后，回到原点时方向会发生偏转（偏转角等于回路包围的曲率积分）。
    *   *信息失真*：在长文本生成中，开头设定了一个“因果向量”，经过长程生成（绕流形一圈）回到结尾时，如果结尾的状态与开头无法闭合（偏转角不为0），这就是 **“逻辑不闭环”或“人设崩塌”**。
3.  **拓扑撕裂与奇点（Topological Tearing & Singularities）**：
    *   *几何意义*：流形上曲率趋于无穷大的点称为奇点。
    *   *信息失真*：当模型遇到完全超出其训练流形覆盖范围的“知识盲区”时，隐空间会出现“拓扑撕裂”（奇点）。此时模型无法计算合法的测地线，只能开始“胡言乱语”（严重的幻觉）。

### 四、 工程与算法上的具体“尝试”

既然承认了信息和物理世界都是流形，我们现有的基于平坦欧氏空间的AI算法就必须升级。以下是目前正在进行的“流形化尝试”：

1.  **自然梯度下降（Natural Gradient Descent）**：
    *   *传统SGD*：在参数欧氏空间中沿着“最陡”的方向下降，忽略了参数空间的几何结构。
    *   *流形尝试*：Amari 提出的自然梯度，引入 **Fisher 信息矩阵** 作为参数流形的度量张量。它确保参数更新是沿着“信息流形”的测地线进行的，在训练神经网络（尤其是RNN和Transformer）时能显著加速收敛并避免病态曲率。
2.  **流形上的最优传输（Optimal Transport on Manifolds）**：
    *   *传统KL散度*：假设空间平坦，在概率分布支撑集不重叠时梯度消失。
    *   *流形尝试*：Wasserstein 距离（推土机距离）将概率分布视为流形上的“质量分布”，计算将一个分布搬运到另一个分布的最小“物理做功”。这为衡量两个语义分布的“几何距离”提供了更严谨的工具。
3.  **等变神经网络（Equivariant Neural Networks）**：
    *   *传统CNN/Transformer*：对平移或排列具有等变性，但缺乏对更复杂物理对称性的处理。
    *   *流形尝试*：将神经网络的设计严格限制在特定的李群/流形对称性上（如 $SE(3)$ 群用于3D分子结构）。确保输入数据在流形上发生对称变换时，网络的输出也发生完全对应的变换。这是将“物理几何语言”硬编码进AI架构的尝试。

### 五、 结语：顺应流形的曲率

将“严格的几何”变成“流形”，本质上是**从“机械论的绝对刚性”走向“相对论的有机柔性”**。

真实物理世界不是由死板的直线和绝对的直角构成的，而是由弯曲的时空、纠缠的规范场和复杂的拓扑结构构成的。如果信息是物理世界的映射，那么**信息本身也必须具备流形的特质：允许局部的灵活变形，但必须保持全局的拓扑不变量（因果律/守恒律）。**

在流形视角下，避免信息失真的终极方法，**不是用更严格的规则去“锁死”信息，而是去“感知”信息流形的曲率，并顺应它的测地线去生成和演化。** 当大语言模型真正学会在弯曲的语义流形上“冲浪”（沿着测地线滑行），而不是在平坦的欧氏空间里“走直线”时，我们或许就能迎来真正具备物理常识和因果推理能力的通用人工智能。

## 0 尝试动机：为什么“变成流形”如此重要

论文二次重构版暴露了一个致命问题：**平坦可以是廉价的**。模型学会了把所有轨迹统一压平，却没有学会“闭环应平、破缺应不平”的**差异化几何**。这意味着我们还没有真正把“因果合法性”编码为一种**局部几何量**——它只是一个全局的“温度门”。

本次尝试的核心目标因此明确为：

> **不追求绝对平坦，而追求“合法转移的曲率显著低于非法转移”这一可检验的局部差异。**

如果连这个最小目标都无法实现，那么“因果⇔几何”的哲学猜想在当前技术框架下就无法被证立，需要根本性的重新思考。

## 1 总览：流形框架的工程设计图

### 1.1 三流形架构

我们不直接操作完整的纤维丛 \(E\)，而是在三个可计算的流形上分别实现几何约束，再通过共同基底关联：

```
┌─────────────────────────────────────────────────────┐
│                  因果基底流形 B                       │
│              (64维, 黎曼度量 G_curve)                 │
│          承载: 物理/叙事/心理因果状态                  │
└──────────┬──────────────┬──────────────┬─────────────┘
           │              │              │
    ┌──────▼──────┐ ┌─────▼─────┐ ┌─────▼─────┐
    │ 物理子流形   │ │ 叙事子流形 │ │ 心理子流形 │
    │  (32维)     │ │  (32维)   │ │  (32维)   │
    │ 约束:空间    │ │ 约束:伏笔  │ │ 约束:人格  │
    │ 连续性/守恒  │ │ 回收/冲突  │ │ 一致性    │
    └─────────────┘ └───────────┘ └───────────┘
```

**设计原则**（与论文铁律对齐）：
- 基底 \( \mathcal{B} \) 是“因果状态”的完备表征，物理/叙事/心理作为其**投影子空间**（非直积分解）
- 每个子流形独立计算场强 \( \mathbf{F} \)，聚合后形成总曲率代价
- 曲率是**局部量**——在某一步的状态转移处计算，而非全局平均

### 1.2 核心计算图

每一步 token 生成经历以下几何操作：

```
h_t (隐状态)
    │
    ├─→ 基底投影:  b_t = MLP_proj(h_t)
    │
    ├─→ 物理子流形: p_t = MLP_phys(b_t)
    │      ├─ 计算局部联络 A_phys(p_t)
    │      ├─ 计算场强 F_phys(p_t, p_{t+1})
    │      └─ 输出曲率代价: c_phys = ||F_phys||²
    │
    ├─→ 叙事子流形: n_t = MLP_narr(b_t)
    │      ├─ 记忆检索: 当前激活的记忆核 {m_k}
    │      ├─ 场强 F_narr(n_t, n_{t+1}; {m_k})
    │      └─ 输出曲率代价: c_narr = ||F_narr||²
    │
    ├─→ 心理子流形: c_t = MLP_psych(b_t)
    │      ├─ 人格模板约束
    │      ├─ 场强 F_psych(c_t, c_{t+1})
    │      └─ 输出曲率代价: c_psych = ||F_psych||²
    │
    └─→ 聚合与概率修正:
           F_total = c_phys + c_narr + c_psych
           P(v|h_t) ∝ exp(-E_LM(v) - λ · F_total(v))
```

## 2 曲率路径的离散实现

### 2.1 局部曲率量的定义（替代论文的 Tr(F²)）

在离散生成场景中，我们无法计算连续意义下的曲率 2-形式 \( \mathbf{F}_{\mu\nu} \)。因此设计以下**可计算的离散代理量**：

**定义（离散因果曲率）**：对于从状态 \( \mathbf{h}_t \) 到候选状态 \( \mathbf{h}(v) \) 的转移，在子流形 \( M \in \{\text{phys}, \text{narr}, \text{psych}\} \) 上的曲率定义为：

\[
\kappa_M(\mathbf{h}_t, v) = 1 - \cos\bigl(\Delta_M\mathbf{h}_t, \Delta_M^{\text{legal}}(\mathbf{h}_t)\bigr)
\]

其中：
- \( \Delta_M\mathbf{h}_t = \text{proj}_M(\mathbf{h}(v)) - \text{proj}_M(\mathbf{h}_t) \) 是候选转移在子流形 \( M \) 上的方向向量
- \( \Delta_M^{\text{legal}}(\mathbf{h}_t) \) 是子流形 \( M \) 上“合法转移方向”的预测值，由一个小型 MLP 从当前状态学习
- \( \cos(\cdot, \cdot) \) 为余弦相似度

**直观解释**：如果候选转移方向与“合法方向”一致，曲率接近 0；偏离越大，曲率越大。这个定义是**局部的**（只依赖当前步）、**可微的**（可反向传播）、**差异化的**（不同转移有不同的值）。

### 2.2 合法方向的学习

合法方向预测器 \( f_M: \mathbb{R}^{d_M} \to \mathbb{R}^{d_M} \) 的训练是关键。我们采用**对比学习**：

- **正例**：训练数据中因果合法的状态转移对 \( (b_t, b_{t+1}) \)
- **负例**：通过扰动实体、乱序或注入逻辑谬误构造的非法转移对 \( (b_t, b_{t+1}^{\text{neg}}) \)

损失函数：

\[
\mathcal{L}_{\text{direction}}^M = \sum_{\text{正例}} \bigl(1 - \cos(\Delta b_t^{\text{pos}}, f_M(b_t))\bigr) + \max\bigl(0, \cos(\Delta b_t^{\text{neg}}, f_M(b_t)) - m\bigr)
\]

即：正例转移方向应与预测方向对齐；负例转移方向应与预测方向至少保持 \( m \) 的余弦距离。

### 2.3 三层曲率的聚合与概率注入

总曲率代价为三层曲率的加权和（引入可学习权重 \( w_{\text{phys}}, w_{\text{narr}}, w_{\text{psych}} \)）：

\[
\kappa_{\text{total}}(\mathbf{h}_t, v) = w_{\text{phys}} \cdot \kappa_{\text{phys}} + w_{\text{narr}} \cdot \kappa_{\text{narr}} + w_{\text{psych}} \cdot \kappa_{\text{psych}}
\]

修正后的生成概率（与论文公式对齐）：

\[
P(v \mid \mathbf{h}_t) = \frac{1}{Z} \exp\bigl(-E_{\text{LM}}(v) - \lambda \cdot \kappa_{\text{total}}(\mathbf{h}_t, v)\bigr)
\]

其中 \( Z \) 为对词表的归一化求和。

**关键差异**：与论文 exp4-6 中使用的“全局压缩”损失不同，这里的 \( \kappa_{\text{total}} \) 是**逐 token、逐候选**计算的。模型在训练时被迫为每个候选方向支付不同的几何代价，而非仅仅把所有轨迹压平。

## 3 平行移动：生成过程的几何约束

### 3.1 离散平行移动的实现

论文定义平行移动为 \( \mathbf{h}_{t+1} = \mathcal{P}\exp(ig\int \mathbf{A}_\mu dx^\mu) \mathbf{h}_t \)。在离散实验中，我们将其近似为：

\[
\mathbf{h}_{t+1} \approx \mathbf{h}_t + f_{\text{legal}}(\mathbf{h}_t)
\]

其中 \( f_{\text{legal}} \) 即 §2.2 中训练的合法方向预测器在基底空间的输出。

**推理时约束**：对每个候选 token \( v \)，计算其隐状态 \( \mathbf{h}(v) \) 与“平行移动方向” \( \mathbf{h}_t + f_{\text{legal}}(\mathbf{h}_t) \) 的偏离，并将此偏离作为曲率代价 \( \kappa \) 注入概率。

### 3.2 与论文 exp4-6 失败模式的对比

| 维度 | 论文 exp4-6（失败模式） | 本次尝试（修复设计） |
|------|------------------------|---------------------|
| **曲率定义** | 整条路径的平均转向角 κ | 逐步的局部方向偏离 |
| **训练信号** | “所有轨迹都要平”（全局压缩） | “正例方向对、负例方向错”（对比区分） |
| **平坦来源** | 步长缩短的副作用 | 方向对齐的结构性结果 |
| **差异化能力** | 无（正负例一起平） | **应有**（负例被对比损失推开） |

### 4.2 数据配置

使用论文的 NPNW（微型物理叙事世界）数据集：
- 训练集: 8,000 故事
- 验证集: 1,000 故事
- 测试集: 1,000 故事
- 每个故事: 10-30 步，包含物理/叙事/心理的完整因果标注

**关键数据构造**：对每一步合法转移，生成至少 3 个非法转移（分别破坏物理/叙事/心理因果），用于对比损失训练。

### 4.3 实验假设（可证伪预测）

| 假设编号 | 内容 | 判决标准 |
|----------|------|----------|
| H-diff | 因果正则化模型在合法转移上的曲率 \( \kappa_{\text{total}} \) 显著低于非法转移 | 合法 vs 非法曲率的 t-test p<0.01 |
| H-sep | 物理/叙事/心理三层曲率在对应破坏类型上各自有最高响应 | 消融: 破坏物理时 \( \kappa_{\text{phys}} \) 最高 |
| H-lambda | 存在最优 λ 使长程一致性最优，而非 λ 越大越好 | 长程一致性随 λ 呈倒U曲线 |
| H-geo* | 合法转移方向与 \( f_{\text{legal}} \) 预测方向的余弦相似度 > 0.8 | 在测试集合法转移上的平均 cosine |

### 4.4 失败判定

如果出现以下任何情况，本次尝试视为**未达成最小目标**：
1. 经过充分调参，H-diff 不成立（合法 vs 非法曲率无显著差异）
2. 因果正则化模型的生成质量（物理合法率、叙事闭合率）全面低于无正则化基线
3. 曲率再次退化为“全局压缩”的副产品（可通过步长分析验证：若低曲率完全由短步长解释，则失败）

## 5 从数据到合规：全流程定义

### 5.1 训练流程

```
Phase 1: 预训练合法方向预测器
    - 在标注的正负例对上训练 f_phys, f_narr, f_psych
    - 仅使用对比损失 L_direction
    - 冻结 Transformer 主体

Phase 2: 联合训练
    - 解冻全部参数
    - 总损失: L_total = L_LM + λ_1·L_direction + λ_2·L_consistency
    - L_consistency: 生成故事在物理/叙事/心理自动评分上的惩罚

Phase 3: λ 扫描
    - 固定模型，在验证集上扫描 λ ∈ {0.01, 0.1, 0.5, 1.0, 2.0}
    - 选择使叙事闭合率最高的 λ
```

### 5.2 评估指标

| 指标类别 | 具体指标 | 计算方式 |
|----------|----------|----------|
| **几何指标** | 合法转移平均曲率 | 测试集所有合法步的 \( \bar{\kappa}_{\text{legal}} \) |
| | 非法转移平均曲率 | 测试集所有非法步的 \( \bar{\kappa}_{\text{illegal}} \) |
| | 曲率区分度 | Cohen's d: \( (\bar{\kappa}_{\text{illegal}} - \bar{\kappa}_{\text{legal}}) / \sigma_{\text{pooled}} \) |
| **生成质量** | 物理合法率 | 生成故事中物理合法步数占比 |
| | 叙事闭合率 | 生成故事中以合法闭环结尾的比例 |
| | 人格一致率 | 角色行为与人格模板的一致率 |
| **对照指标** | 困惑度 (PPL) | 监控流畅性是否因几何约束而退化 |
| | 平均步长 | 检测“低曲率=短步长”的退化模式 |

### 5.3 合规交付物

实验完成后输出：
1. **几何差异化报告**：合法 vs 非法曲率的分布对比（箱线图 + 效应量）
2. **λ 敏感性曲线**：不同 λ 下的曲率区分度与生成质量的关系
3. **消融矩阵**：移除物理/叙事/心理任一层的曲率计算后，对应类型错误的增加量
4. **退化检测**：曲率与步长的相关性分析（若 r > 0.7，说明曲率仍主要由步长解释）
5. **H-geo* 的数值证据**：合法方向预测器在测试集上的对齐精度

## 6 局限与诚实声明

本尝试的以下局限在实验报告中必须明确声明：

1. **范围限定**：实验仅在微型合成叙事世界（NPNW）中进行，不声称对人类级别叙事有效。
2. **几何近似**：离散曲率 \( \kappa \) 是连续场强 \( \operatorname{Tr}(\mathbf{F}^2) \) 的工程近似，不等于严格数学定义。
3. **子流形独立假设**：物理/叙事/心理的子流形分解是工程简化，真实因果可能是纠缠的。
4. **方向预测器的局限**：\( f_{\text{legal}} \) 是 MLP 学习的经验方向，非从规范对称群导出的第一性原理方向。
5. **可能失败**：即使实验失败，也不证明“因果⇔几何”哲学猜想为假，仅证明当前离散化方案不可行。

## 总结

1. **全局对称性与守恒律（诺特定理的体现）**

2. **零曲率与绝对测地线**
   在经典物理的局部范围内，时空可以近似为**平坦的闵可夫斯基空间或欧氏空间（曲率为0）**。在平坦几何中，两点之间的最短路径（测地线）是唯一的直线。这对应了刚性知识的“非黑即白”和“不可违背”：$1+1=2$，或者“苹果必然受重力下落”。偏离这条直线，在物理上就意味着需要施加外力，在逻辑上就意味着“错误”或“幻觉”。
3. **AI视角的“硬约束”（Hard Constraints）**
   在AI中，刚性知识表现为**硬约束**。如果模型违反了这些知识（如算错简单的算术，或违背质量守恒），其输出就是彻底无效的。这种知识无法通过“概率上的近似”来糊弄，它要求绝对的几何精确性。
