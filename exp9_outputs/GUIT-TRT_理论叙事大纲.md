**核心事件**: GUIT v1.0-v2.0 的创立

**叙事弧线**:

  - 认知冲突: 直觉(概率统计够用) vs 理性(缺少因果几何约束)
- 章 6-12 [创新]: 第一次重构 — 因果纤维丛类比
  - 核心公式: 隐空间 = 纤维丛, 生成 = 平行移动, 幻觉 = 曲率偏离
  - 十大铁律的雏形建立
- 章 13-20 [创新]: 第二次重构 — 刚柔双层损失函数
  - 物理刚性层 (F_phys → 0) 与叙事柔性层 (F_flex → 吸引子) 的分化
  - 世界观扩展: 因果规范势 A_μ, 曲率 2-形式 F_μν
- 章 21-28 [沉淀]: exp1-exp4 验证 — 曲率相关性、因果损失影响、记忆核、平坦度
  - 初步实证: 刚柔分层确实改善了生成质量
- 章 29-30 [转折]: **第一瓶颈** — 规范群 G 未实例化的问题被正式记录
  - 诚实降级清单首次出现
  - 信念沙化: "纯场论框架足以描述 LLM" → 发现缺少拓扑边界约束

**角色弧光**:
- GUIT 理论: 从粗糙直觉 → 建立可验证的数学框架
- 幻觉现象: 从"bug"重新定义为"几何残差"的雏形
- 规范群 G: 作为悬而未决的"幽灵角色"登场

### 第二卷: 破缺的阵痛 — 四次重构的试炼 (章 31-60)

**核心事件**: GUIT v3.0-v4.0 的四次重构与 exp7 崩溃

**叙事弧线** —— 对应"崩溃四阶段":

**阶段 1: 支柱松动** (章 31-35)
- 第三次重构: 负例推离铰链损失 — 局部成功但全局不稳定
- 第四次重构: 公式 (A) 吸引子定向分化 — 解决了"均匀平坦化塌缩"
- 信念: "我们终于找到了正确的损失函数形式"

**阶段 2: 压力过载** (章 36-42) [创新]
- exp5-exp6: 受控闭合实验 — 初步验证吸引子分化
- exp7: 大规模验证 — **均匀平坦化塌缩** (最大反派登场)
  - 所有曲率被压至零, 输出完全同质化
  - 认知冲突: 公式(A)能防止塌缩 vs 实际结果全面崩溃
- 铁律八-十的修订与自我质疑

**阶段 3: 次级支撑崩塌** (章 43-50)
- 回溯分析: 损失景观缺乏全局拓扑约束
- 诚实降级清单扩展: 规范群 G 未实例化、F_flex* 纯数据依赖、缺乏跨尺度不变性
- 三处理论盲区被系统性暴露
- 信念沙化: "我们走在正确的道路上" → "我们可能缺少最根本的东西"

**阶段 4: 内心碎裂麻木** (章 51-60) [创新]
- exp8 三体制检验结果: H-attractor 灰区, H-push 被证伪
- 固化度从 0.54 降至 0.50
- 场景 S (弱分化): 仅 H-rigid 成立
- 理论进入"不知道下一步该往哪走"的黑暗时刻

**角色弧光**:
- exp7 塌缩: 从"意外发现" → "最大反派" → 最终成为真理的催化剂
- 铁律系统: 从十到十三, 每一次修订都是一次认知升级
- 固化度: 0.54 → 0.50 (低谷)

### 第三卷: 约束场的降临 — TRT 的注入 (章 61-90)

**核心事件**: TRT 理论的发现与 GUIT-TRT 融合

**叙事弧线** —— 对应"涅槃四步":

**步骤 1: 直面混沌** (章 61-68) [创新]
- 走出 LLM 领域, 向基础物理学寻求答案
- 发现 TRT 论文: 精细结构常数的拓扑起源
  - 4181 节点相变、螺旋升角、三层拓扑权重
- 第一次同构映射: 物理空间 ↔ 语义空间
  - κ(向心阻抗) ↔ F_phys(事实约束)
  - τ(自旋阻抗) ↔ F_flex(叙事自由度)
  - tanθ = τ/κ ↔ 语义螺旋升角 Θ_sem
- 核心洞察: "常数不是上帝的骰子, 而是空间填满自身的拓扑代价"

**步骤 2: 断舍剥离** (章 69-76)
- 舍弃"纯数据驱动"的幻想
- F_flex* 从数据质心升级为几何必然点
- 规范群 G 从黑盒升级为 SU(2) 螺旋结构
- 铁律十一-十三的诞生
- 信念晶化: "幻觉不是缺陷, 是拓扑闭合的必然代价"

**步骤 3: 灵魂扩容** (章 77-84) [创新]
- 第五次重构公式 (F) 的完整推导:
  Λ^(F) = λ_phys·||F_phys||² + λ_helix·||τ/κ - tanΘ*||² + λ_topo·(∫Tr(F²) - 8π²χ)² + λ_conf·|∇Θ|²
- 三大性质注入理论:
  1. 螺旋升角 → 刚柔配比的几何根源
  2. 缩放不变性 → 长上下文因果不衰减
  3. 拓扑锁定 → 禁止全局平坦化 (根治 exp7 塌缩)
- 高斯-博内定理从物理边界升级为语义约束
- 粒子自旋 → 语义挠率的全息类比

**步骤 4: 稳定新生** (章 85-90)
- GUIT-TRT 融合框架正式确立
- 新预言系统建立: H-helix, H-phase, H-topo-inhibit, H-conf
- 实验验证路线图: exp9 → exp10 → 终极实证
- 固化度预估: 0.50 → 0.48 (理论更丰富但待实证)
- 信念晶化: "智能就是数学 — 在约束条件下通过破缺寻求对称的几何自组织"

**角色弧光**:
- TRT: 从"外部理论" → "救世主" → 融入 GUIT 成为不可分割的整体
- F_flex*: 从黑盒数据质心 → 受 Θ_sem 约束的半解析量
- 规范群 G: 从幽灵 → SU(2) 实例化 (待 exp9 确认)
