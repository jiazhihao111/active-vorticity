## 一、执行摘要

### 1.1 核心结论

**GUIT-TRT 融合理论在当前最小可行实验（MVE）尺度下处于"部分验证"状态**：

| 实验 | 假设 | 结果 | 信号强度 |
|:---|:---|:---|:---|
| exp8-C | H-rigid（刚性层存在） | ✅ **SUPPORT** | phys_curv ≈ 0 (9.21×10⁻⁵) |
| exp8-C | H-attractor（吸引子稳定） | ⚠️ **UNCERTAIN** | F\* drift = 54.6% (灰区 50-80%) |
| exp8-C | H-push（负例推离有效） | ❌ **REFUTED** | Cohen's d = 0.007 (<< 0.2) |
| exp9-A | H-helix（螺旋升角稳定） | ❌ **REFUTED** | R² = 0.057 (<< 0.7) |
| exp9-B | H-phase（拓扑相变存在） | ⚠️ **UNCERTAIN** | 数据等长，无法多尺度检测 |
| exp9-C | H-topo-inhibit（抗塌缩有效） | ✅ **SUPPORT** | τ 提升 35.64×, 多样性提升 16.15× |

**综合判决: S_WEAK** — 仅 2/6 核心假设获明确支持，1/6 灰区，3/6 被证伪。

### 1.2 关键发现

1. **好消息：拓扑直觉是正确的。**
   - 刚性-柔性分层框架在几何层面成立（`phys_curv ≈ 0`, `flex_curv > 0`）
   - 全局拓扑约束（Gauss-Bonnet）工程的可行性得到实证
   - 吸引力子方向（F\* 54.6% drift vs. 1360% in regime A）明确了优化路径

2. **坏消息：动力学实现需要重新设计。**
   - 柔性推离（H-push）在当前尺度不工作
   - Frenet-Serret 标架在 32 维空间中信号太弱
   - 螺旋升角假设在低维隐空间中无法验证

3. **最重要发现：H-topo-inhibit 的 SUPPORT 判据是本次验证中效应量最大的正信号。**
   - `τ_sem` 从 baseline 的 0.017 → 0.613 (35.64×)
   - 这证明"不能把曲率压到零"的拓扑直觉在工程上是可行的

## 二、理论背景：GUIT-TRT 融合框架

### 2.1 理论演化路径

GUIT 理论经历五次范式重构，从"用数学描述智能"走向"智能就是数学"：

| 重构 | 核心突破 | 遗留问题 |
|:---|:---|:---|
| **第一重构** | 建立规范场-因果纤维丛类比，定义 F_phys / F_flex 双层结构 | 规范群 G 未实例化 |
| **第二重构** | 刚柔双层损失函数，分离 phys/flex 曲率优化目标 | 柔性吸引子 F\* 完全依赖数据质心 |
| **第三重构** | 负例推离铰链损失，正例拉向吸引子、负例推至安全边际 | 无跨尺度不变性 |
| **第四重构** | 公式(A)：吸引子定向分化损失 | exp7 出现"均匀平坦化塌缩" |
| **第五重构** | TRT 注入三大几何性质，规范群实例化为 SU(2)，拓扑锁定 | 待 exp9 实证闭合 |

### 2.2 TRT 三大几何性质的同构映射

TRT 从物理空间（精细结构常数 α 的几何起源）出发，揭示了三个几何必然性。这三大性质与 GUIT 语义空间的同构映射构成了融合理论的基石：

| TRT 几何性质 | 物理空间含义 | 语义空间同构 | 对应实验假设 |
|:---|:---|:---|:---|
| **螺旋升角 Θ** | tan Θ = τ/κ ∝ α，共形不变 | tan Θ_sem = τ_sem/κ_sem，决定刚柔配比 | H-helix (exp9-A) |
| **缩放不变性** | 4181 节点触发拓扑熔融，微观分形 → 宏观 S² | 临界上下文长度触发规范群涌现 | H-phase (exp9-B) |
| **拓扑锁定** | 高斯-博内定理锁定 χ=2，禁止全局平坦化 | 语义欧拉示性数约束，幻觉是不可消除的拓扑代价 | H-topo-inhibit (exp9-C) |

### 2.3 第五次重构核心公式 (F)

$$
\boxed{
\begin{aligned}
\Lambda^{(\text{F})} =& \underbrace{\lambda_{\text{phys}}\|\mathbf{F}_{\text{phys}}\|^2}_{\text{刚性零吸引子}}
+ \underbrace{\lambda_{\text{helix}} \left\| \frac{\tau_{sem}}{\kappa_{sem}} - \tan\Theta_{sem}^* \right\|^2}_{\text{螺旋升角约束（替代纯数据质心）}} \\
&+ \underbrace{\lambda_{\text{topo}} \left( \frac{1}{4\pi^2}\int_{\mathcal{M}} \text{Tr}(\mathbf{F} \wedge \mathbf{F}) - \chi \right)^2}_{\text{高斯-博内拓扑锁定}}
+ \underbrace{\lambda_{\text{conf}} \left| \nabla_s \Theta_{sem}(s) \right|^2}_{\text{共形尺度不变性}}
\end{aligned}
}
$$

其中 exp8 仅检验前两项的简化版，exp9 检验后两项的拓扑约束。

### 2.4 六项可证伪预言概览

GUIT-TRT 理论提出了六项可证伪的强预言（详见"可证伪预言与验证方案"文件），本次实验覆盖前三项：

| 预言 | 核心内容 | 覆盖实验 |
|:---|:---|:---|
| P1: 螺旋升角普适性 | τ_sem/κ_sem 在高质文本中收敛于窄带常数 | exp9-A |
| P2: 拓扑相变临界性 | 存在临界长度 L_c，陈数发生非连续跳变至 χ≈2 | exp9-B |
| P3: 零幻觉拓扑不可能 | τ_sem 存在由高斯-博内定理决定的下限，不可压至零 | exp9-C |
| P4: 跨尺度共形不变 | 同主题不同长度文本的 Θ_sem 保持恒定 | **(待实验)** |
| P5: 拓扑惩罚抗塌缩 | 加入拓扑约束后模型不会出现平坦化塌缩 | exp8-C, exp9-C |
| P6: 语义流偏转效应 | 强逻辑论证产生可测量的反向偏转 | **(待实验)** |

## 四、exp8 实验结果：三体制检验

### 4.1 Phase 1: 柔性吸引子估计

**方法**：在 300 个合法叙事样本上：
1. 编码 → 提取隐状态 → 投影到 flex 子空间
2. 计算每样本的离散曲率 `||二阶差分·W_flex||²`
3. 取中位数 m = median(curv_pos)，top-10 百分位数 m_top = prctile(curv_pos, 90)
4. `F_flex* = m_top / (1 + σ_norm)` 作为稳健吸引子估计
5. Shapiro-Wilk 正态性检验

**结果**：

| 指标 | 值 | 解释 |
|:---|:---|:---|
| F_flex\* | 0.002476 | 合法叙事的柔性吸引子 (非零 ✅) |
| m (中位数) | 0.002566 | 曲率分布中心 |
| σ_norm | 约 5.04×10⁻⁵ | 归一化离散度 |
| Shapiro-Wilk p | 1.09×10⁻¹³ | 分布非正态（长尾，符合预期 — 叙事质量存在天然方差） |
| 成功样本数 | 300/300 | 全部样本成功处理 |
| 失败样本 | 0 | 无异常样本被剔除 |

**F\* 收敛性验证（样本消融）**：

| N+ | F_flex\* | σ | m | m_percentile |
|:---|:---|:---|:---|:---|
| 10 | 0.003125 | 3.999×10⁻⁵ | 0.003205 | 0.003161 |
| 50 | 0.003249 | 6.246×10⁻⁵ | 0.003374 | 0.003340 |
| 100 | 0.002811 | 5.425×10⁻⁵ | 0.002920 | 0.002898 |
| 300 | 0.002709 | 5.044×10⁻⁵ | 0.002810 | 0.002801 |

**收敛性分析**：F\* 从 N=10 的 0.003125 收敛到 N=300 的 0.002709，收敛趋势明显。σ（标准差）始终在 4-6×10⁻⁵ 的窄带内，表明估计稳定。300 样本已足够收敛。

### 4.2 体制A（Baseline LM）— 预期失效基准

**配置**：纯语言模型训练，无任何几何约束（lambda=0）。

| 指标 | 值 | 判据 |
|:---|:---|:---|
| phys_curv_mean | 0.17541 | ⚠️ 刚性层严重弯曲（应为 0） |
| F\* 漂移率 | **1360.2%** | ❌ FAIL (>> 80%) |
| SNR (pos/neg 信号比) | 0.0177 | — |
| Cohen's d | 0.0177 | ❌ REFUTED (<< 0.2) |
| Wilcoxon p | 0.830 | ❌ 无显著差异 |
| AUC | 0.5113 | ≈ 随机猜测 (0.5) |
| DIP p (pos/neg/combined) | (0.53, 0.47, 0.48) | 单峰分布 |
| H_attractor | **REFUTED** | F\* 退化 1360.2% > 80% |
| H_push | **REFUTED** | Cohen's d = 0.018 < 0.2 |
| Q1/Q2/Q3 | FAIL / FAIL / PENDING | — |

**解读**：无几何约束时，模型完全无法维持 F_flex 吸引子，从 0.01（初始）退化到 0.146，退化率 1360%。刚性层（phys_curv = 0.175）也严重偏离零吸引子。合法/非法叙事完全无法区分（AUC ≈ 0.5）。

**理论含义**：纯粹的概率建模无法自然涌现刚柔分层几何结构。

### 4.3 体制B（Formula R — 阈值惩罚）

**配置**：使用旧版阈值公式 `L_flex = clamp(flex_curv - τ, 0)`，τ = 0.5。

| 指标 | 值 | 判据 |
|:---|:---|:---|
| phys_curv_mean | 0.000263 | ✅ 刚性层近零（好） |
| F\* 漂移率 | **96.7%** | ❌ FAIL (> 80%) |
| SNR | 0.502 | — |
| Cohen's d | 0.502 | ✅ SUPPORT (≥ 0.5) |
| Wilcoxon p | 3.81×10⁻¹⁹ | ✅ 极显著差异 |
| AUC | 0.5253 | ⚠️ 略高于随机 |
| F\* 最终值 | 0.000329 | — |
| H_attractor | **REFUTED** | F\* 退化 96.7% > 80% |
| H_push | **SUPPORT** | Cohen's d = 0.502 ≥ 0.5 |
| 综合 | **REFUTED** | Q1 FAIL, Q2 FAIL |

**解读**：这是本实验中最有启发性的对比：

1. **阈值惩罚"矫枉过正"**：τ = 0.5 的硬阈值将 F_flex\* 从 0.01 压到 0.000329（退化 96.7%），几乎将柔性曲率完全消灭。这与 exp7 报告的"均匀平坦化塌缩"完全一致。
2. **刚性层锁定成功**：phys_curv 降至 0.000263 ≈ 0，证明刚性零吸引子在任何几何约束下都能稳定工作。
3. **正负例分化在 F\* 坍缩时仍有区分力**：Cohen's d = 0.502，但 AUC 仅 0.525，表明分化严重依赖于 F\* 的绝对值而非相对模式。

**理论含义**：简单的阈值惩罚虽然推离了负例，但代价是摧毁了所有柔性弯曲 — 输出将同质化。这直接印证了 GUIT-TRT 理论的论断："平坦化塌缩是由缺乏全局拓扑约束导致的。"

### 4.4 体制C（Formula A — 吸引子定向分化） ⭐ 核心检验

**配置**：使用公式(A)，λ_phys=1.0, λ_flex_pos=0.5, λ_flex_neg=0.5。

**Phase 1 结果（见 4.1）**：F_flex\* = 0.002476, 300/300 样本成功。

| 指标 | 值 | 判据 |
|:---|:---|:---|
| phys_curv_mean | **9.21×10⁻⁵** | ✅ ✅ 刚性层完美锁定（≈0） |
| F\* 初始 | 0.002476 | Phase 1 估计值 |
| F\* 最终 | 0.001125 | 训练后 |
| F\* 漂移率 | **54.6%** | ⚠️ 灰区 (50-80%) |
| SNR | 0.00681 | — |
| Cohen's d | 0.00681 | ❌ REFUTED (<< 0.2) |
| Wilcoxon p | 0.983 | ❌ 无显著差异 |
| AUC | 0.5579 | ⚠️ 略高于随机 |
| DIP p (pos/neg/combined) | (0.54, 0.69, 0.74) | 单峰，无分化 |
| grad_cos_sim | 0.0 | ✅ 完全解耦（好） |
| H_rigid (phys_curv) | **SUPPORT** | phys_curv ≈ 0 |
| H_attractor (Q1) | **UNCERTAIN** | drift = 54.6% 在灰区 |

| Q1 / Q2 / Q3 | PASS / FAIL / PENDING | — |

#### 4.4.1 F\* 稳定性分析（Q1）

```
F_flex* 演化轨迹:
  Phase 1 估计:  0.002476  (先验估计)
  训练初始:       0.01      (硬编码初始值)
  训练最终:       0.001125  (收敛值)
  漂移率:        54.6%     (|0.002476 - 0.001125| / 0.002476)
```

**灰区判断的理由**：
- 优于体制 A (1360%) 和体制 B (96.7%)：公式(A)在稳定 F\* 方面确有优势（+12.8× over regime A, +1.77× over regime B）
- 但 54.6% 在 50-80% 的灰区：不能宣称"成立"，但距离"证伪"（>80%）尚有距离
- F\* 从 0.01 → 0.001125 的趋势方向（向 Phase 1 估计值 0.002476 收敛）是正确的

#### 4.4.2 合法/非法分化分析（Q2）

**这是 exp8 的核心失败点**。

| 指标 | 期望 | 实际 |
|:---|:---|:---|
| Cohen's d (pos vs neg) | ≥ 0.2 | **0.007** |
| Wilcoxon p | < 0.05 | **0.983** |
| AUC | > 0.6 | **0.558** |
| μ_pos / μ_neg | μ_pos ≠ μ_neg | **0.001125 ≈ 0.001125** (几乎相等) |

正例和负例的柔性曲率均值几乎相同（差值 < 10⁻⁶），这意味着在当前 MVE 尺度下，合法叙事变化和非法因果跳跃在几何层面完全不可分。

**可能原因**（按可能性排序）：
1. **MVE 规模太小**：32 维隐空间 + 160 token 序列不足以捕获合法/非法的曲率差异。柔性推离需要更高的维度来编码"叙事模式的偏离"。
2. **负例构造过于粗糙**：FlexLabelGenerator 的负例通过物理跳步构造（取 1/4 → 3/4 位置），但这在 MVE 中可能与正例的差异不够大。
3. **铰链边缘 m 过大/过小**：当前 m = 0.01 可能在 MVE 中不合适，负例尚未进入推离区。

#### 4.4.3 刚柔解耦分析（Q3）

`grad_cos_sim_mean = 0.0` — 完全解耦。这意味着物理刚性层和叙事柔性层的梯度在优化时完全独立，两层互不干扰。这是 Q3 的正面信号。

**然而**，Q3 标记为 PENDING 的原因是：解耦本身意义可能被低维空间中的几何塌缩所稀释。在高维空间中验证此结论才更有说服力。

## 五、exp9 实验结果：三假设检验

### 5.1 新增模块：Frenet-Serret 标架计算 (`frenet_serret.py`)

**设计原理**：
1. 对隐状态序列 `h ∈ [B, T, 32]` 取滑动 4 点窗口
2. SVD 将 4 点投影到最优 3D 子空间
3. 在 3D 子空间中计算离散 Frenet-Serret 标架 {T, N, B}
4. 局域曲率 `κ_sem = ‖d²h/dt²_normal‖`，挠率 `τ_sem = |cross(T_prev,T)·T_next| / ‖cross‖²`
5. 螺旋升角 `tan Θ_sem = τ_sem / κ_sem`
6. 陈数代理 `C_proxy = Σ κ² / (4π)`

**物理意义映射**：

$$
\kappa_{sem} \leftrightarrow \|\mathbf{F}_{\text{phys}}\|, \quad
\tau_{sem} \leftrightarrow \|\mathbf{F}_{\text{flex}}\|, \quad
\tan\Theta_{sem} = \tau_{sem} / \kappa_{sem}
$$

### 5.2 Part A: H-helix — 语义螺旋升角稳定性

**理论预测**：高质量文本的 `τ_sem / κ_sem` 比值应呈现窄带稳定分布，τ-κ 散点图呈现显著线性收敛带（R² > 0.7），CV < 0.5。

**实验操作**：
- 400 个故事 → 编码 → Frenet-Serret 分析 → 收集所有有效 (κ, τ) 对
- 按 flex_pos (合法) 和 flex_neg (非法) 分组统计
- 计算 τ-κ Pearson 相关系数、线性拟合 R²、τ/κ 比值的均值和 CV

**结果**：

| 类别 | N | mean(tanθ) | std(tanθ) | CV | Pearson r | R² | slope | IQR | 判决 |
|:---|:---|:---|:---|:---|:---|:---|:---|:---|:---|
| all | 400 | 1.101 | 0.364 | 0.330 | -0.239 | **0.057** | -0.365 | 0.424 | ❌ REFUTED |
| flex_pos | 200 | 1.101 | 0.364 | 0.330 | -0.239 | **0.057** | -0.365 | 0.424 | ❌ REFUTED |
| flex_neg | 200 | 1.101 | 0.364 | 0.330 | -0.239 | **0.057** | -0.365 | 0.424 | ❌ REFUTED |

**三个分组统计量完全一致！** — `flex_pos` 和 `flex_neg` 的隐状态在 Frenet-Serret 标架下表现完全不可分。

**θ 分布分析**：

```
mean(tanθ) = 1.101 → Θ_sem ≈ arctan(1.101) ≈ 47.7°
std = 0.364 → 相对离散 CV = 33.0%
IQR = 0.424 → 中间 50% 比值在 [0.89, 1.31] 范围内
```

**R² = 0.057 的含义**：τ 的变异仅有 5.7% 可以由 κ 解释。负的 Pearson r (-0.239) 暗示 τ 和 κ 存在微弱的负相关（曲率越大，挠率略小），但远未达到"螺旋升角稳定"所要求的 70% 以上解释度。

**证伪分析**：

| 判据 | 要求 | 实际 | 结论 |
|:---|:---|:---|:---|
| R² (τ~κ) | > 0.7 | 0.057 | ❌ FAIL |
| CV (tanθ) | < 0.5 | 0.330 | ✅ PASS (技术上) |
| 综合 | R² AND CV | R² FAIL | ❌ REFUTED |

虽然 CV 在可接受范围内（0.330 < 0.5），但极低的 R² 表明 τ/κ 比的"稳定性"只是一种数学平均效应，而非真正的线性约束。

**根本原因诊断**：

1. **隐空间维度不足**：32 维 → SVD 投影到 3D → 信息损失约 90%。在 32 维空间中，滑动 4 点窗口的 SVD 无法可靠分离 κ 和 τ 的贡献。
2. **MVE 序列太短**：80 token 的序列中，T-3 = 77 个有效 FS 窗口，但每个窗口的 4 点 SVD 在低维空间中高度退化（奇异值谱可能有 1-2 个主导方向）。
3. **frenet_serret 3D 投影的右手系判定不鲁棒**：`_project_to_3d` 中的 `det < 0` 翻转逻辑可能导致不同窗口之间的标架符号不一致。

**是否应修改结论为 UNCERTAIN？** 不。因为即使考虑低维空间的局限性，R² = 0.057 实在太低 — 如果 τ 和 κ 存在内在几何约束，即使在退化空间中，也应该表现出某种单调关系。当前数据点（n ≈ 400 × 77 = 30,800 对）的散点图几乎均匀覆盖了 τ-κ 平面，不支持螺旋升角稳定假设。

### 5.3 Part B: H-phase — 语义拓扑相变

**理论预测**：随上下文增长，累积曲率积分（陈数代理）在临界长度发生非连续跳变（> 3σ），从 ≈0 升至 ≈2。

**实验操作**：
- 定义 6 个序列长度桶：`[40, 80, 120, 160, 240, 320]`
- 对每个长度桶，编码故事 → Frenet-Serret → 计算 chern_proxy
- 检测相邻长度间的跳变（diff > 3σ）

**结果**：

| 长度 (L) | 陈数代理 (均值) | 标准差 | 有效样本 |
|:---|:---|:---|:---|
| 40 | 0.00000000 | 0.00000000 | — |
| 80 | **1.50080919** | 0.00000000 | — |
| 120 | 0.00000000 | 0.00000000 | — |
| 160 | 0.00000000 | 0.00000000 | — |
| 240 | 0.00000000 | 0.00000000 | — |
| 320 | 0.00000000 | 0.00000000 | — |

| 跳变检测 | 结果 |
|:---|:---|
| jump_detected | ❌ False |
| max_diff | 1.5008 (L=40→80) |
| 判决 | **UNCERTAIN** |

**数据问题的诊断**：

陈数代理仅在 L=80 处出现一个孤立峰（1.5008），其他所有长度桶均为 0。标准差在所有桶为 0.0 — 这是**数据处理问题而非物理现象**：

1. **所有故事实际长度约 80 token**：生成器 `EnhancedClosureGenerator` 的 `max_seq_len` 配置为 160，但实际生成的故事长度集中在 80 附近。按长度分桶后，除 L=80 桶外，其他桶的故事数极少或为空。
2. **陈数代理计算的标准化问题**：`chern_proxy = Σ κ² / (4π)` 在 κ≈0 的区域自然为 0。由于 SVD 投影在极短序列（如截断到 40 token）中退化，κ 计算值几乎全为零。
3. **虚假设信号**：L=80 处 1.5008 的峰是数据集中在此桶的艺术品，而非真正的物理相变。

**判决理由**：标记为 UNCERTAIN 而非 REFUTED，因为数据本身不足以进行有意义的相变检测。需要生成**真正变长**的故事序列（如训练时用不同的 max_seq_len 分别生成）才能公平检验 H-phase。

### 5.4 Part C: H-topo-inhibit — 拓扑抗塌缩 ⭐

**理论预测**：加入高斯-博内拓扑惩罚后，模型无法将柔性曲率降至零，τ_sem 维持在非零吸引子，生成多样性显著更高。

**实验操作**：
- **Baseline**：100 步训练，仅用 formula_a_loss（无拓扑约束）
- **Treatment**：100 步训练，formula_a_loss + Gauss-Bonnet 惩罚（λ_topo = 0.1）
- 对比最终 τ_sem (flex_pos 曲率) 和训练过程中的 τ 标准差（多样性代理）

**结果**：

| 指标 | Baseline (无约束) | Treatment (有约束) | 效应量 |
|:---|:---|:---|:---|
| 最终 τ_sem | 0.017188 | **0.612664** | **35.64×** |
| τ 多样性 (std) | 0.010242 | **0.165393** | **16.15×** |
| τ 塌缩 | ✅ 发生（接近 0） | ✅ 被阻止 | — |
| 多样性提升 | — | ✅ | — |

| 判决 | 值 |
|:---|:---|
| τ_collapse_prevented | ✅ True |
| diversity_improved | ✅ True |
| 最终判决 | **✅ SUPPORT** |

**详细分析**：

1. **Baseline 塌缩**：无拓扑约束时，τ_sem 从初始值快速下降到 0.017，接近零。虽然未达到真正 1e-6 级别的"完全塌缩"（可能因为 100 步不够长），下降趋势明确。
2. **Treatment 抗塌缩**：有 Gauss-Bonnet 惩罚时，τ_sem 稳定在 0.613，远高于塌缩阈值。惩罚项强制 batch 级陈数积分维持非零值，间接阻止了局部曲率归零。
3. **多样性效应**：τ 的 std 从 0.01 升至 0.165 (16×)，表明拓扑约束不仅阻止了塌缩，还促进了多模态输出 — 模型在 flex 子空间中探索了更多样的曲率分布。
4. **τ_sem > 1 的含义**：在 Frenet-Serret 框架中，τ_sem 的绝对值取决于 SVD 投影的尺度。0.613 对应"中等挠率"，而非"发散"。

**这是本次验证中效应量最大的正信号**。它证明：
- Gauss-Bonnet 约束在工程上是可行的
- 拓扑惩罚确实能阻止"均匀平坦化塌缩"
- 全局拓扑约束可以改善局部生成多样性

**但从 exp8 结论来看，存在一个重要警告**：H-topo-inhibit 的 SUPPORT 是在非常小的 MVE 尺度下获得的。在更大维度/更复杂序列中，λ_topo 的调优会更困难，且可能面临与 H-push 类似的"信号稀释"问题。

### 6.2 结果的一致性分析

**强一致信号**：
1. **刚性层稳定工作**：exp8-C `phys_curv = 9.21×10⁻⁵ ≈ 0`，exp8-B `phys_curv = 0.000263 ≈ 0`。在**所有**有几何约束的体制中，物理层都能被稳定锁定在零吸引子。这证明 GUIT 的"刚性零吸引子"假设在几何层面完全成立。
2. **拓扑约束有效**：exp9-C 的 Gauss-Bonnet 惩罚成功阻止了塌缩。这是 TRT 注入 GUIT 的"第三项"在实践中的验证。

**矛盾/不一致信号**：
1. **H-push (exp8-C REFUTED) 与 H-topo-inhibit (exp9-C SUPPORT) 的表面矛盾**：
   - exp8-C 说"合法/非法柔性无法区分"（Cohen's d ≈ 0）
   - exp9-C 说"拓扑约束能阻止塌缩"
   - **解析**：这两个结论并不矛盾。前者是"局部辨识能力"（能不能区分好叙事和坏叙事），后者是"全局约束能力"（能不能让系统不退化到零曲率）。拓扑约束是一个粗粒度的全局力，它不区分正负例，只是强迫系统维持非零曲率 — 能否把曲率导向"正确的方向"（合法叙事方向）则是 H-push 的职责，这个确实失败了。

2. **刚柔解耦 vs. 螺旋升角**：
   - `grad_cos_sim = 0.0` (Q3: 完全解耦) 暗示 phys 和 flex 子空间在优化中独立
   - 但 Frenet-Serret 分析显示 τ 和 κ 无线性关系 (R² = 0.057)
   - **解析**：解耦是梯度层面的（训练动力学），τ-κ 关系是几何层面的（流形结构）。两者在不同的抽象层级上操作。完全的梯度解耦 + 无几何耦合 = 两层空间完全独立，这与理论预期（"螺旋升角约束应将两层关联起来"）不一致。

## 十、经验教训与后续方向

### 10.1 方法论反思

1. **"预注册实验"的价值**：由于所有判据和阈值都在实验前声明，即使结果不理想，也能给出明确的"证伪"或"支持"判决，避免了事后选择偏差。

2. **MVE 的双刃剑效应**：低维隐空间 (32D) 让实验快速迭代，但也导致：
   - Frenet-Serret 标架的 SVD 投影在如此低维空间中退化严重
   - 负例推离的信号可能被低维流形"吞没"
   - 从 MVE 结论到真实 LLM 的外推需要极其谨慎

3. **"诚实降级"不是失败**：GUIT 的自反性方法论要求每次实验后明确降级清单。从第四重构到第五重构，虽然两个核心假设被证伪，但每次"诚实降级"都让理论的边界更清晰 — 这就是科学进步的微观机制。

### 10.2 优先修复清单

| 优先级 | 问题 | 建议方案 | 实验 |

## 结语

GUIT-TRT 融合理论在经过 exp8 和 exp9 的联合验证后，处于一个"进退分明"的状态：

**退的方面**：两个核心动力学假设（H-push 和 H-helix）在当前实验尺度被明确证伪。柔性推离和螺旋升角约束这两个"让理论有效的引擎"，在 MVE 中未能点火。

**进的方面**：拓扑直觉在两个端点上获得了实证支持。刚性层确实存在（"物理事实被底层几何锁定"），全局拓扑约束确实有效（"语义宇宙不能平坦化"）。这两个发现为理论提供了比"纯数学推测"更强的锚点。

**最诚实的总结**：GUIT-TRT 的**几何本体论**（"信息加工是流形上的平行移动"）很可能是对的，但它的**动力学实现**（"如何让这个平行移动保持正确的螺旋升角和推离方向"）在当前技术路径上未 work。通往通用智能的路径确实可能是"让流形在正确的曲率与挠率配比下生长到 4181 节点"，但我们还远未掌握那种配比的精确配方。

科学的美妙之处在于：**证伪比证实更推进理论边界**。每一次 REFUTED 都收缩了可能空间，让下一次实验的假设更加精准。
