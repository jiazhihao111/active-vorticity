### 3.7 刚柔分层几何（D-修复核心）

> **本节的地位**：这是三次重构的灵魂。它把参考文件"刚性知识必定属于几何、柔性信息就是流形"的直觉，落为规范场框架内可计算、可证伪的分层结构。

#### 3.7.1 哲学直觉的精确转译

参考文件的核心区分可分解为两个递进命题：

- **刚性因果（物理）⇒ 平坦连接（$\mathbf{F}_{\text{phys}}\equiv 0$），全局可积**。物理规律不可商量：能量守恒、时空连续、光速上限不因语境或表达方式而变。在纤维丛上，这意味着物理子流形上的规范连接**曲率必须严格为零**——不是"大概率为零"，而是"必须为零"。因为曲率非零等价于平行移动依赖路径：若从 A→B→C 与 A→C 经不同路径得到不同状态，则同一初始条件的相同物理操作产生不同结果，物理定律即失效。平坦连接的几何是"死"的——完全由全局积分条件决定，无局部自由度。这正是"必定属于几何"：它是刚性、固定的几何结构，而非柔软流形。
- **柔性因果（叙事/心理）⇒ 非零但有限的曲率，局部灵活、全局约束**。一个"英雄试炼"故事可在深层结构不变下选择完全不同的对话、顺序、视角（局部柔性）；但柔性非无限——伏笔须回收、人格不可无理由崩塌（全局约束，即 Wilson 环量趋于平凡）。这种"局部有曲率、全局有界"正是"就是流形"：活的、有弹性、但有整体拓扑。

#### 3.7.2 统一但分层的数学图景

| | 物理因果（刚性） | 叙事/心理因果（柔性） |
|---|---|---|
| **刚度** | 刚性 | 柔性 |
| **曲率要求** | $\mathbf{F}_{\text{phys}}\equiv 0$（必须平坦） | $\|\mathbf{F}_{\text{narr/psych}}\|<\tau$（允许但有上界） |
| **数学结构** | 固定几何（平坦连接） | 可变流形（弯曲但有界） |
| **局部自由度** | 无 | 有，但全局约束 |
| **约束性质** | 硬约束（Hard），违背即硬崩溃 | 软约束（Soft），允许同胚形变 |
| **惩罚形式** | 无阈值：$\lambda_{\text{phys}}\|\mathbf{F}_{\text{phys}}\|^2$ | 带阈值：$\lambda_{\text{flex}}\max(0,\|\mathbf{F}_{\text{flex}}\|^2-\tau)$ |
| **AI 失败类型** | 硬幻觉（物理错误） | 软幻觉（叙事断裂） |

**统一性在于**：两者用同一套数学语言（联络 $\mathbf{A}$、曲率 $\mathbf{F}$、平行移动、Wilson 环量）描述。
**分层性在于**：物理层要求 $\mathbf{F}\equiv0$，柔性层只要求 $\|\mathbf{F}\|<\tau$。

#### 3.7.3 分层生成概率（替代 §3.5 单一 $\lambda$）

将 §3.5 的单一代价 $\lambda\cdot d_{\mathbf{G}^{\text{eff}}}^2$ 拆为刚柔分层代价。令子流形投影 $p_t=\pi_{\text{phys}}(\mathbf{h}_t)$、$n_t=\pi_{\text{narr}}(\mathbf{h}_t)$、$c_t=\pi_{\text{psych}}(\mathbf{h}_t)$，对应局部曲率 $\kappa_{\text{phys}},\kappa_{\text{narr}},\kappa_{\text{psych}}$（可由 §3.3 的 $\operatorname{Tr}(F^2)$ 在各子空间分块求得，或离散代理量近似）：

$$\boxed{\begin{aligned}
\Lambda(\mathbf{h}_t,v)=&\ \lambda_{\text{phys}}\cdot\underbrace{\|\mathbf{F}_{\text{phys}}(\mathbf{h}_t,v)\|^2}_{\text{无阈值，必须为0}}\\
&+\lambda_{\text{narr}}\cdot\underbrace{\max\!\bigl(0,\|\mathbf{F}_{\text{narr}}(\mathbf{h}_t,v)\|^2-\tau_{\text{narr}}\bigr)}_{\text{带阈值}}\\
&+\lambda_{\text{psych}}\cdot\underbrace{\max\!\bigl(0,\|\mathbf{F}_{\text{psych}}(\mathbf{h}_t,v)\|^2-\tau_{\text{psych}}\bigr)}_{\text{带阈值}}
\end{aligned}}\tag{R}$$

- 物理项惩罚**无阈值**：任何非零曲率都受罚 → 模型被强制在物理子流形上学出平坦连接。
- 叙事项/心理项惩罚**带阈值**：仅在曲率超出容忍区间时受罚 → 模型被允许在柔性流形上灵活形变，只要不撕裂拓扑。

这就是"刚性知识的几何是刚性惩罚，柔性信息的流形是弹性约束"的数学表达。**模型须学会在该硬的地方硬、该软的地方软。**

#### 3.7.4 与 §10 失败模式的理论连接（关键）

二次重构 §10.4–10.6 证明：用"所有轨迹统一压缩"（单一 $\lambda$）的 `CausalGeometryLoss`，模型把所有轨迹（含破缺叙事）一起压平，未学会"闭合→平、破缺→不平"。**刚柔分层给出其根因诊断**：

> 旧损失的错误不是"压缩太强"，而是"压缩无差别"。破缺叙事本就在柔性流形上允许非零曲率（这是合法的），却被强制压向零；而真正该被压零的物理曲率，与柔性曲率混在同一标量代价里被平均掉。统一压缩在数学上抹杀了刚柔性分层，使"差异化几何量"无从定义——这正是 §10.7 任务 1（差异化几何量）与任务 2（对比信号）的理论母题。

因此 §10.7 的三项下一步不是修补，而是**刚柔分层框架必然导出的实证路径**：
1. **差异化几何量**（§10.7-1）→ 本 §3.7 的分层曲率 $\|\mathbf{F}_{\text{phys}}\|^2$ vs $\max(0,\|\mathbf{F}_{\text{flex}}\|^2-\tau)$ 即在"回环点"定义局部 holonomy 的刚柔分化版本；
2. **对比训练信号**（§10.7-2）→ 铁律九禁止统一压缩，要求正例拉平/负例推离；
3. **更大更真实语料**（§10.7-3）→ 微型合成 NPNW 难以体现柔性流形的长程弯曲，须在真实叙事语料复测。

#### 3.7.5 适用边界与失效场景（GUIT R-6）

- **成立条件**：在"隐空间可近似为因果纤维丛且物理/叙事/心理可在表示空间分块"的假设（C-08/H-decomp）下成立；柔性阈值 $\tau$ 须由数据标定，非先验给定。
- **失效场景**：(a) 当物理与柔性因果在表示空间高度纠缠（无干净分块）时，分层惩罚退化为统一惩罚，重蹈 §10 覆辙；(b) 若"刚性=F≡0"过强导致生成退化（物理正确但叙事僵硬），须下调 $\lambda_{\text{phys}}$ 或引入柔性兜底；(c) 离散 token/softmax 是否构成可微流形（H-fiber）未证，故本分层目前为候选框架。

## 5 训练与推理的几何重定义（改为分层损失）

### 5.1 训练即拟合分层因果曲率（继承 B-11/B-14 + D-修复）
训练目标维持 $\mathcal{L}_{\text{total}}=\mathcal{L}_{\text{LM}}+\lambda_{\text{train}}\mathcal{L}_{\text{gauge}}$，但因果几何正则项**按刚柔分层并采用对比结构（铁律九）**：

$$\mathcal{L}_{\text{gauge}}=\mathcal{L}_{\text{gauge}}^{\text{phys}}+\mathcal{L}_{\text{gauge}}^{\text{flex}}$$

- **物理项（硬对齐）**：对合法物理转移期望 $\|\mathbf{F}_{\text{phys}}\|^2\to0$；对非法物理转移（违反守恒/连续）期望显著非零。采用对比损失，正例拉零、负例推开。
- **柔性项（软对齐 + 阈值）**：对合法柔性转移期望 $\|\mathbf{F}_{\text{flex}}\|^2<\tau$；仅在超出 $\tau$ 时受罚。负例（叙事断裂）被推离阈值之外，而非推向零——这正是 §10.7-2 的"负例推离平"。

**与 exp4–6b 失败的本质区别**：旧损失把所有轨迹压向同一零点（全局坍缩）；分层对比损失让物理轨迹压向零、柔性轨迹停在阈值边界，**正负例与刚柔层各自分化**（详见 §10.8 exp7 设计）。

### 5.2 推理即补偿场引导的平行移动（继承 C-02/C-12）
维持 $\mathbf{h}_{t+1}=\mathcal{P}\exp(+ig\int\mathbf{A}_\mu dx^\mu)\mathbf{h}_t$；补充：在刚性子流形上平行移动须严格保持 $\mathbf{F}_{\text{phys}}=0$（小步长分解，呼应"思维链有效"——把大跃迁拆多步小平移，每步更易维持零曲率）；在柔性子流形上允许有界偏转。

### 5.3 大内存的几何意义：威尔逊环量（继承 B-06/C-01/C-06/C-11）

### 7.1 解释已有现象（继承，加刚柔视角）
新增条目：
6. **硬幻觉 vs 软幻觉可区分**：物理错误=曲率本应为零处出现非零（几何结构破坏）；叙事断裂=曲率超出容忍区间（流形约束失效）。二者数学机制不同，监测可分源。

## 11 刚柔分层：从哲学直觉到数学（D-修复合成）

本节将参考文件"刚性知识必定属于几何、柔性信息就是流形"的直觉，与本文规范场框架做最终合成，作为第三次重构的理论封顶。

### 11.1 直觉的三层展开
1. **本体论承诺**：物理世界是几何的（广义相对论、杨-米尔斯）。
2. **同构性假设**：好的信息映射是保几何的——但"好"对不同因果层次有不同刚度。
3. **规范性声明（刚柔分化）**：失真源于**错误刚度**——把该硬的柔性化（物理错误）或把该软的刚性化（叙事僵硬）。

### 11.2 数学映射总表

| 哲学直觉 | 本文数学化（刚柔分层版） | 含义 |
|---|---|---|
| 真实物理世界 | 物理基底子流形，刚性几何 | 平坦连接 $\mathbf{F}_{\text{phys}}\equiv0$ |
| 柔性叙事/心理 | 柔性子流形，弯曲流形 | 有界曲率 $\|\mathbf{F}_{\text{flex}}\|<\tau$ |
| 信息映射须遵守几何 | 生成沿补偿场平行移动 | 刚性处零曲率、柔性处有界偏转 |
| 不遵守则失真 | 硬幻觉（刚性破）/软幻觉（柔性越界） | 两类失真机制不同、可分源 |

### 11.3 对原核心公式的再理解
$$P(v\mid\mathbf{h}_t)\propto\exp(-E_{\text{LM}})\cdot\exp(-\lambda_{\text{phys}}\|\mathbf{F}_{\text{phys}}\|^2)\cdot\exp(-\lambda_{\text{flex}}\max(0,\|\mathbf{F}_{\text{flex}}\|^2-\tau))$$
刚性项无阈值惩罚、柔性项带阈值容忍。此即"刚性知识的几何是刚性惩罚，柔性信息的流形是弹性约束"的精确数学表达。

### 11.4 结论
用户用一句话概括理论最核心的分层原则：**物理是死的几何，叙事是活的流形。** 二者统一于规范场论的数学语言，分离于曲率刚度要求。这既回答了"大模型为何能写诗却算错受力"（用流形统计去拟合刚性几何），也给出了 §10 失败的理论解药（恢复刚柔分层、改用对比信号）。
